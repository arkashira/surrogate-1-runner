# Quick Start Guide for CodeMuse Extension

## Installation

1. Open Visual Studio Code.
2. Go to the Extensions view by clicking on the Extensions icon in the Activity Bar on the side of the window or pressing `Ctrl+Shift+X`.
3. Search for "CodeMuse" in the Extensions Marketplace.
4. Click on the "Install" button next to the CodeMuse extension.

## Usage

Once installed, you can activate the CodeMuse extension by opening any text editor in VS Code. The extension will automatically provide AI-powered coding suggestions based on the context of your code.

To manually trigger the extension, you can use the command palette (`Ctrl+Shift+P`) and type "CodeMuse: Hello World" to see a demonstration message.

## Features

- AI-powered coding suggestions
- Context-aware assistance
- Seamless integration with VS Code
