
## One-Click Conda Environments for Axentx Surrogate-1

This directory contains three pre-configured Conda environment YAML files that bundle the dependencies needed for:

1. **Open-Weight LLMs** – The core language models used by Surrogate-1.
2. **Benchmark Suite** – Evaluation scripts and datasets for model comparison.
3. **SDK** – The Axentx SDK that provides helper utilities for data ingestion and model training.

1. **Prerequisites**
