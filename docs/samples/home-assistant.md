# Home Assistant Sample

## Overview
This sample demonstrates how to integrate the Axentx Surrogate-1 sensor with Home Assistant.

## Repository
The sample repository is available at: [https://github.com/axentx/samples/tree/main/home-assistant](https://github.com/axentx/samples/tree/main/home-assistant)

## Configuration
The sample includes a pre-configured `configuration.yaml` file with the necessary settings to connect to the Axentx Surrogate-1 sensor.
