# Introduction to Support
The axentx surrogate-1 project provides a dedicated support channel for users to ask questions and get help with the migration process. This guide outlines the support resources available and how to access them.

## Support Channel
The support channel is available on Slack. To join the channel, please follow these steps:

1. Go to the [Slack invite link](https://example.com/slack-invite-link)
2. Click on the "Join" button
3. Fill out the registration form with your email address and password
4. Verify your email address by clicking on the link sent to you by Slack

## Support Documentation
This support guide is available at `/opt/axentx/surrogate-1/docs/support_guide.md`. It provides an overview of the support resources available and how to access them.

## Response Time
The response time to support requests is within 24 hours. If you have any questions or need help with the migration process, please don't hesitate to reach out to us on the Slack channel.

## Contact Information
If you have any questions or need help with the migration process, please contact us at [support@axentx.com](mailto:support@axentx.com).
