# Surrogate-1 Documentation

Welcome to the Surrogate-1 documentation. This guide covers everything you need to know about using the Surrogate-1 parallel public-dataset ingest workers.

## Getting Started

### Quick Start

Follow these steps to get started with Surrogate-1:

1. Clone the repository
2. Install dependencies
3. Configure your environment
4. Run the ingest workers
