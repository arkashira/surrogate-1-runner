   * Alerts are logged at `WARNING`; normal operation logs at `INFO`; failures at `ERROR`.

6. **Looping**  
   * When `--interval` is supplied, the main loop sleeps for the specified number of seconds between checks.  
   * Without `--interval`, the script performs a single check and exits.

   * Tests for the default `fetch_bulk_credits` should confirm it raises `NotImplementedError`.  
   * Tests for `default_notifier` should capture the warning log.  
   * Tests for `send_alert_sns` should mock `boto3` to verify the publish call and the log message.
