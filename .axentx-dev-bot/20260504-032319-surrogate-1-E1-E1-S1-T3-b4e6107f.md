| **Singleton per‑integration** – `get_logger(integration_name)` lazily creates (and caches) an `APILogger` instance, so callers do not need to manage lifetimes. | Simple, memory‑efficient API. |
| **Fallback to the standard `logging` module** – the internal `_write_entry` uses the same lock that the `logging` handler uses, so if a user prefers the richer `logging` ecosystem they can enable it with a single flag. | Gives flexibility without code duplication. |
| **Graceful JSON serialization** – a small `_safe_serialize` helper that falls back to `str()` for non‑serializable objects. | Guarantees the logger never crashes because of a weird payload. |

"""
Thread‑safe, JSON‑line logger for sandboxed API traffic.

* Writes one JSON object per line to a file under the sandbox root.
* Environment variables:
    - SANDBOX_ROOT          : base directory for sandbox artefacts
    - SANDBOX_API_LOG_PATH  : full path to the log file (overrides the default)
* Provides `log_request`, `log_response` helpers and a per‑integration
  singleton accessor `get_logger(integration_name)`.
"""

from __future__ import annotations

import json
import logging
import os
import threading
from datetime import datetime
from typing import Any, Dict, Optional

# ----------------------------------------------------------------------
# 1️⃣  Configuration & directory preparation
# ----------------------------------------------------------------------
_SANDBOX_ROOT = os.getenv("SANDBOX_ROOT", "/opt/axentx/surrogate-1/sandbox")
_DEFAULT_LOG_DIR = os.path.join(_SANDBOX_ROOT, "logs")
_DEFAULT_LOG_FILE = os.path.join(_DEFAULT_LOG_DIR, "api_requests.log")

# Allow a full‑path override (useful for tests or custom deployments)
_LOG_PATH = os.getenv("SANDBOX_API_LOG_PATH", _DEFAULT_LOG_FILE)

# Ensure the directory hierarchy exists before any write happens.
os.makedirs(os.path.dirname(_LOG_PATH), exist_ok=True)

# ----------------------------------------------------------------------
# 2️⃣  Low‑level thread‑safe writer (used by both the custom writer and the
#     standard logging handler)
# ----------------------------------------------------------------------
_write_lock = threading.Lock()


def _safe_serialize(data: Any) -> str:
    """
    Convert *data* to a JSON string.
    Non‑serializable objects are stringified so logging never raises.
    """
    try:
        return json.dumps(data, default=str, ensure_ascii=False)
    except Exception:  # pragma: no cover – extremely defensive
        return json.dumps(str(data), ensure_ascii=False)
