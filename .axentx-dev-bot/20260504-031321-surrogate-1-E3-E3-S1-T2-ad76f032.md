## Usage Examples & Best Practices for the Surrogate‑1 Validation Engine

The Surrogate‑1 Validation Engine is a lightweight Java library that validates
structured data against a set of rules defined in JSON Schema or custom
validation DSL.  Below you will find step‑by‑step examples that cover:

1. **Project setup** – adding the library to your Maven/Gradle build.
2. **Basic validation** – validating a single JSON payload.
3. **Batch validation** – validating a stream of records.
4. **Custom rule integration** – extending the engine with your own logic.
5. **Best practices** – performance, error handling, and versioning tips.

> **Note**: All examples assume you are using Java 17 or newer.

---

### 1. Project Setup

#### Maven

```xml
<dependency>
  <groupId>com.axentx.surrogate</groupId>
  <artifactId>surrogate-validation</artifactId>
  <version>1.4.2</version>
</dependency>
```

#### Gradle (Kotlin DSL)

```kotlin
implementation("com.axentx.surrogate:surrogate-validation:1.4.2")
```

> The library is published to Maven Central. If you use a private repository,
> add the appropriate repository block.

### 2. Basic Validation

```java
import com.axentx.surrogate.validation.Validator;
import com.axentx.surrogate.validation.ValidationResult;
import com.axentx.surrogate.validation.exceptions.ValidationException;

public class BasicExample {
    public static void main(String[] args) {
        // Load the validator from a JSON Schema file
        Validator validator = Validator.fromSchemaPath("schemas/user.json");

        String jsonPayload = """
            {
              "id": 123,
              "name": "Alice",
              "email": "alice@example.com"
            }
            """;

        try {
            ValidationResult result = validator.validate(jsonPayload);
            if (result.isValid()) {
                System.out.println("✅ Payload is valid!");
            } else {
                System.out.println("❌ Validation failed:");
                result.getErrors().forEach(err -> System.out.println(" - " + err));
            }
        } catch (ValidationException e) {
            System.err.println("Validation engine error: " + e.getMessage());
        }
    }
}
```

### 3. Batch Validation

For high‑throughput ingestion pipelines, validate a stream of JSON objects.

```java
import com.axentx.surrogate.validation.Validator;
import com.axentx.surrogate.validation.ValidationResult;
import java.io.BufferedReader;
import java.io.FileReader;
import java.util.stream.Stream;

public class BatchExample {
    public static void main(String[] args) throws Exception {
        Validator validator = Validator.fromSchemaPath("schemas/order.json");

        try (BufferedReader br = new BufferedReader(new FileReader("orders.jsonl"))) {
            Stream<String> lines = br.lines();
            lines
                .map(validator::validate)
                .filter(result -> !result.isValid())
                .forEach(result -> {
                    System.err.println("Invalid record:");
                    result.getErrors().forEach(err -> System.err.println(" - " + err));
                });
        }
    }
}
