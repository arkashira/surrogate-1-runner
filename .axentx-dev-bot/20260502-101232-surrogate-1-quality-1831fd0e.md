## Final Synthesis (Best of Both Candidates)

**Diagnosis (merged, de-duplicated)**
- Hot-path authenticated HF API calls (load_dataset / list_repo_tree) cause intermittent 429s and non-reproducible runs.
- No persisted file-list cache; each run re-enumerates repos, wasting quota and changing shard ordering.
- Lightning Studio reuse is fragile: idle timeout kills training; scripts burn quota by recreating instead of reusing/restarting.
- Ingestion produces mixed-schema parquet (extra `source`, `ts` columns) instead of strict `{prompt,response}` and filename-based attribution, risking pyarrow CastError downstream.
- No deterministic shard-to-repo routing or sibling-repo spread; concurrent ingestion can hit HF 128/hr/repo limit.

**Proposed change (merged)**
- Embed a persisted file-list cache (JSON) generated once on the Mac orchestrator.
- Modify training to use CDN-only fetches, enforce Lightning Studio reuse with idle restart, and use deterministic shard ordering.
- Tighten ingestion schema projection to strict `{prompt,response}` plus filename-based attribution, and route shards across sibling repos to respect HF commit caps.

Usage (Mac orchestrator):
  1) Generate file list once (outside rate-limit window):
     HF_REPO=datasets/your-org/surrogate-1 python train.py --gen-filelist 2026-04-29
  2) Launch/attach studio and run training:
     FILELIST_PATH=filelist.json python train.py
"""

from typing import List, Optional

import lightning as L
import torch
from torch.utils.data import Dataset, DataLoader
import requests
from tqdm import tqdm

HF_REPO = os.getenv("HF_REPO", "datasets/your-org/surrogate-1")
FILELIST_PATH = pathlib.Path(os.getenv("FILELIST_PATH", "filelist.json"))
CDN_BASE = f"https://huggingface.co/datasets/{HF_REPO}/resolve/main"
SIBLINGS = int(os.getenv("SIBLINGS", "5"))

# ---- File-list generation (run once on Mac) ----
def gen_filelist(date_folder: str = "2026-04-29", out_path: pathlib.Path = FILELIST_PATH) -> List[str]:
    """
    Run once (or after rate-limit clears) from Mac.
    Requires HF_TOKEN in env for a single list call.
    Returns sorted deterministic list.
    """
    try:
        from huggingface_hub import HfApi
    except ImportError:
        subprocess.check_call([sys.executable, "-m", "pip", "install", "huggingface_hub"])
        from huggingface_hub import HfApi

    api = HfApi()
    entries = api.list_repo_tree(repo_id=HF_REPO, path=date_folder, recursive=True)
    files = sorted(
        e.rfilename for e in entries
        if e.type == "file" and e.rfilename.endswith(".parquet") and not e.rfilename.startswith(".")
    )
    out_path.parent.mkdir(parents=True, exist_ok=True)
    out_path.write_text(json.dumps(files, indent=2))
    print(f"Wrote {len(files)} files to {out_path}")
    return files

# ---- CDN-only dataset ----
class CDNParquetDataset(Dataset):
    """
    Loads parquet shards via CDN. Caches parsed DataFrames per shard in memory.
    Strict projection to {prompt, response} + filename attribution.
    """
    def __init__(self, filelist: List[str], max_rows_per_shard: Optional[int] = None):
        self.filelist = filelist
        self.max_rows_per_shard = max_rows_per_shard
        self._cache = {}

    @staticmethod
    def _load_parquet_cdn(rel_path: str, max_rows: Optional[int] = None):
        url = f"{CDN_BASE}/{rel_path}"
        r = requests.get(url, timeout=30)
        r.raise_for_status()
        import io, pandas as pd
        df = pd.read_parquet(io.BytesIO(r.content))
        if "prompt" not in df.columns or "response" not in df.columns:
            raise ValueError(f"Missing prompt/response in {rel_path}")
        df = df[["prompt", "response"]].copy()
