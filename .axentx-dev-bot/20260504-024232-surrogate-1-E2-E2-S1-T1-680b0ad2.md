## Unified, Production‑Ready Design for API‑Drift Detection  

Below is a single, coherent implementation that merges the strongest ideas from **Candidate 1** and **Candidate 2** while removing the overlap and contradictions.  

* **What the module does**  
  1. **Collects** the current API‑signature snapshot (`Signature` structs).  
  2. **Loads** the stored baseline for a given service from Redis.  
  3. **Compares** the two sets, producing a drift flag and a human‑readable diff.  
  4. **Persists** a `DriftEvent` (via a pluggable persister) only when drift is detected.  

* **Why this is better than either candidate alone**  

| Feature | Candidate 1 | Candidate 2 | Unified |
|---------|-------------|-------------|---------|

| **Signature comparison** | Straight‑forward map‑based diff, helper `headersEqual`, `joinDiffs` | None | Keep the full diff logic from Candidate 1 (it already covers new, removed, and header‑drift cases). |
| **Error handling & context** | Returns wrapped errors, uses context everywhere | Same | Preserve. |
| **Testability** | Interface‑based design, easy to mock | No interfaces | Keep the interfaces. |

package detector // /opt/axentx/surrogate-1/src/detector

import (
	"context"
	"time"
)

// Signature – immutable description of one API endpoint.
type Signature struct {
	Method  string            `json:"method"`  // e.g. "GET"
	Path    string            `json:"path"`    // e.g. "/api/v1/users"
	Headers map[string]string `json:"headers"` // canonicalised header → value
}

// DriftEvent – persisted record of a drift detection.
type DriftEvent struct {
	Timestamp time.Time `json:"timestamp"` // UTC
	Service   string    `json:"service"`   // logical service name
	Details   string    `json:"details"`   // human‑readable diff
}

// BaselineRetriever – fetches the stored baseline for a service.
type BaselineRetriever interface {
	// GetBaseline returns the slice of signatures that were stored as the
	// baseline for the given service.  If no baseline exists an empty slice
	// and a nil error are returned.
	GetBaseline(ctx context.Context, service string) ([]Signature, error)
}

// DriftPersister – persists a DriftEvent wherever the caller wants
// (Postgres, ClickHouse, a file, etc.).
type DriftPersister interface {
	Persist(ctx context.Context, ev DriftEvent) error
}

// DetectAndPersist is the single entry‑point used by the detector runner.
// It returns nil on success (including the “no‑drift” case) or a wrapped error.
func DetectAndPersist(
	ctx context.Context,
	service string,
	current []Signature,
	baselineRetriever BaselineRetriever,
	persister DriftPersister,
) error
```
