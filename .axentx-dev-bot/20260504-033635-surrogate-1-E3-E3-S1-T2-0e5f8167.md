import java.util.concurrent.atomic.AtomicBoolean;

public class PauseResume {
    private AtomicBoolean paused = new AtomicBoolean(false);

    public void pause() {
        paused.set(true);
    }

    public void resume() {
        paused.set(false);
    }

    public boolean isPaused() {
        return paused.get();
    }
}

- **Atomic state toggle** – `PauseResume.isPaused()` returns the current value of an `AtomicBoolean` that is flipped by `pause()` and `resume()`.
- **Delegation from Agent** – The abstract `Agent` class forwards calls to `pause()`, `resume()`, and `isPaused()` to its internal `PauseResume` instance.
- **Subclass pause awareness** – A concrete agent subclass (`ExampleAgent`) can query `isPaused()` inside its `run()` loop and skip work when paused.
- **Thread‑safety** – Concurrent calls to `pause()`/`resume()` from multiple threads do not corrupt the flag state; the final state matches the last operation.
- **Package consistency** – All classes (`PauseResume`, `Agent`, `ExampleAgent`) reside in `com.axentx.surrogate1.agents`.
