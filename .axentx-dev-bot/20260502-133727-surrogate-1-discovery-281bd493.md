     This removes the need to list the repo at runtime and guarantees any runner can recompute its slice identically.

     Use this list for all subsequent runs that day. No `/api/` calls during training/ingest.

   - Project to `{prompt, response}` at parse time. Do **not** add `source`, `ts`, or other metadata columns to the row schema.  
     Keep attribution in the filename pattern: `batches/public-merged/{date}/{slug}.parquet` (or `.jsonl` if sharded per-run).

3. **Rate-limit safety and reversibility**  
   - All ingestion uses CDN URLs (`https://huggingface.co/datasets/.../resolve/main/...`).  
   - Fallback to API list only for local/manual runs (clearly marked as not for heavy parallel use).
   - Pre-flight script exits non-zero on 429 and prints `Retry-After`; caller/cron can wait.

import json, sys
from huggingface_hub import HfApi

def main(repo_id, date_folder, out_path):
    api = HfApi()
    entries = api.list_repo_tree(repo_id=repo_id, path=date_folder, recursive=False)
    files = []
    for e in entries:
        if e.type == "directory":
            try:
                subentries = api.list_repo_tree(repo_id=repo_id, path=e.path, recursive=False)
                for se in subentries:
                    if se.type == "file":
                        files.append({"path": se.path, "size": se.size})
            except Exception as exc:
                print(f"Skipping {e.path}: {exc}", file=sys.stderr)
        elif e.type == "file":
            files.append({"path": e.path, "size": se.size if hasattr(se, 'size') else None})
    with open(out_path, "w", encoding="utf-8") as f:
        json.dump(files, f, indent=2)
    print(f"Wrote {len(files)} files to {out_path}")

if __name__ == "__main__":
    main(sys.argv[1], sys.argv[2], sys.argv[3])
PY
