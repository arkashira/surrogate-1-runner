- **Average Price Calculation** – The computed average price for each competitor‑category pair must equal the arithmetic mean of all non‑null, non‑zero prices in that group, rounded to two decimal places.

- **Input Validation** – The function must throw a descriptive error if the input array contains fewer than **3 categories** or fewer than **5 competitors** in total.  
- **Edge‑Case Handling** –  
  - Empty input array → return an empty result set without throwing.
