# SDK Documentation

## Overview

The axentx SDKs provide a set of libraries for popular programming languages to facilitate integration with the axentx pipeline. These SDKs allow developers to interact with the pipeline programmatically, enabling seamless integration into their applications.

### Supported Languages

- Python
- JavaScript (Node.js)
- Java
- Go

## Installation

### Python

To install the Python SDK, run the following command:

```bash
pip install axentx-sdk-python
```

### JavaScript (Node.js)

To install the JavaScript SDK, run the following command:

```bash
npm install axentx-sdk-js
```

### Java

To install the Java SDK, add the following dependency to your `pom.xml`:

```xml
<dependency>
    <groupId>com.axentx</groupId>
    <artifactId>axentx-sdk-java</artifactId>
    <version>1.0.0</version>
</dependency>
```

### Go

To install the Go SDK, run the following command:

```bash
go get github.com/axentx/sdk-go
```

## Usage

### Python

```python
from axentx_sdk_python import PipelineClient

client = PipelineClient(api_key="your_api_key")
response = client.run_pipeline(input_data)
print(response)
```

### JavaScript (Node.js)

```javascript
const { PipelineClient } = require('axentx-sdk-js');

const client = new PipelineClient('your_api_key');
client.runPipeline(inputData)
    .then(response => console.log(response))
    .catch(error => console.error(error));
```

### Java

```java
import com.axentx.sdk.java.PipelineClient;

public class Main {
    public static void main(String[] args) {
        PipelineClient client = new PipelineClient("your_api_key");
        String response = client.runPipeline(inputData);
        System.out.println(response);
    }
}
```

### Go

```go
package main

import (
    "fmt"
    "github.com/axentx/sdk-go"
)

func main() {
    client := sdk.NewPipelineClient("your_api_key")
    response, err := client.RunPipeline(inputData)
    if err != nil {
        fmt.Println(err)
        return
    }
    fmt.Println(response)
}
```

## API Reference

For detailed API reference, please refer to the official documentation for each language-specific SDK.

# CLI Tool Documentation

## Overview

The axentx CLI tool provides a command-line interface for interacting with the axentx pipeline. It supports basic pipeline operations, allowing developers to manage and monitor their pipelines directly from the command line.

## Installation

To install the axentx CLI tool, run the following command:

```bash
npm install -g axentx-cli
```

## Usage

### Run Pipeline

To run a pipeline, use the following command:

```bash
axentx run --input <input_data>
```

### List Pipelines

To list all available pipelines, use the following command:

```bash
axentx list
```

### Get Pipeline Status

To get the status of a specific pipeline, use the following command:

```bash
axentx status --pipeline <pipeline_id>
```

### Stop Pipeline

To stop a running pipeline, use the following command:

```bash
axentx stop --pipeline <pipeline_id>
```

## Configuration

The axentx CLI tool can be configured using a configuration file located at `~/.axentx/config.json`. The configuration file should contain the following properties:

```json
{
    "api_key": "your_api_key",
    "endpoint": "https://api.axentx.com"
}
```

## API Reference

For detailed API reference, please refer to the official documentation.
