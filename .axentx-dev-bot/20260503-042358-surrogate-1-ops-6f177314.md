## Final Implementation Plan (≤2h)

**Highest-value improvement**: Replace fragile shell-based ingestion with a **manifest-driven, CDN-bypass Python worker** that eliminates HF API rate limits during training data loads and prevents mixed-schema CastErrors.

### Changes

1. **`bin/dataset-enrich.sh`** → **`bin/dataset-enrich.py`**  
   - Single API call (post-rate-limit window) to `list_repo_tree(path, recursive=False)` for one date folder → save `manifest.json`.  
   - Worker reads manifest, downloads each file via **CDN bypass** (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`) with no Authorization header.  
   - Stream-parse each file, project to `{prompt, response}` only, compute md5, dedup via central SQLite, emit `shard-N/date/slug.parquet`.  
   - No `load_dataset(streaming=True)` on heterogeneous repos → avoids pyarrow CastError.

2. **`lib/dedup.py`** (unchanged API, optional perf)  
   - Keep SQLite-backed `seen(md5)` check; add batch `executemany` for speed.

3. **`.github/workflows/ingest.yml`**  
   - Matrix `shard_id: [0..15]`.  
   - Each job runs `python bin/dataset-enrich.py --shard $SHARD_ID --date $YYYYMMDD`.  
   - No recursive `list_repo_files`; use folder-level tree calls only.  
   - On 429: wait 360s then retry (single retry policy).  
   - Output to `batches/public-merged/{date}/shard{N}-{HHMMSS}.parquet`.

4. **`requirements.txt`**  
   - Add `requests`, keep `datasets`, `huggingface_hub`, `pyarrow`, `numpy`.

5. **`train.py`** (optional companion)  
   - Embed pre-listed file paths from `manifest.json`; use CDN-only `wget`/`requests` in `IterableDataset` → zero API calls during training.

def list_date_folder(date_str: str):
    """Single API call: folder-level tree (non-recursive)."""
    try:
        tree = API.list_repo_tree(
            repo_id=HF_REPO,
            path=date_str,
            repo_type="dataset",
        )
        return [t.path for t in tree if t.type == "file"]
    except Exception as e:
        print(f"list_repo_tree failed: {e}", file=sys.stderr)
        return []

def save_manifest(date_str: str, files):
    out = Path("manifest") / f"{date_str}.json"
    out.parent.mkdir(exist_ok=True)
    out.write_text(json.dumps(files, indent=2))
    return out

def download_cdn(path: str, dest: Path):
    url = f"{BASE_CDN}/{path}"
    r = requests.get(url, timeout=60)
    r.raise_for_status()
    dest.write_bytes(r.content)
    return dest
