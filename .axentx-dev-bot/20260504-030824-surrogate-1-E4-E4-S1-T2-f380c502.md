# Surrogate‑1 Python SDK

The Surrogate‑1 Python SDK provides a convenient, typed interface to the
Surrogate‑1 REST API.  It bundles request helpers, response models, and
utility functions that make it easy to integrate Surrogate‑1 data into
your own pipelines.

## Installation

```bash
pip install surrogate-1
```

## Quick start

```python
from surrogate import costs

# Retrieve the latest cost anomalies
anomalies = costs.get_cost_anomalies()

# `anomalies` is a list of `Anomaly` objects
for anomaly in anomalies:
    print(f"ID: {anomaly.id}")
    print(f"Description: {anomaly.description}")
    print(f"Severity: {anomaly.severity}")
    print(f"Detected at: {anomaly.detected_at}")
    print("-" * 40)
```

The `get_cost_anomalies()` method internally calls the `/v1/costs/anomalies`
endpoint and returns a list of parsed `Anomaly` model instances.

## Available modules

- `surrogate.costs` – Cost‑related endpoints (including `get_cost_anomalies`)
- `surrogate.datasets` – Dataset ingestion helpers
- `surrogate.models` – Pydantic models for API responses
- `surrogate.utils` – Miscellaneous utilities

## Contributing

how to submit patches, run tests, and report issues.
