## 3. Integration tests

**Happy path tests:**
1. User sets language preference to Spanish, receives Spanish digest
2. User with no preference gets English digest by default
3. User updates language preference, new digests reflect updated language
4. Multiple users maintain separate language preferences
5. Language preference persists through service restart

**Edge case tests:**
1. User submits invalid language code, system rejects and logs error
2. Database connection failure during preference retrieval returns default language
3. User preference is empty string, system falls back to default language
