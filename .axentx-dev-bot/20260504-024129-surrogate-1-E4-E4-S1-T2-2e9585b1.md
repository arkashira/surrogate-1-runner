    "outcome": "string"
  }

```

#### Fields  

| Field | Description | Type |
|-------|-------------|------|

| `simulation_details.duration_seconds` | Length of the simulation in seconds. | number |
| `simulation_details.parameters` | Key‑value pairs of the parameters the user supplied (e.g., contribution amount, age). | object |
| `simulation_details.outcome` | Result of the simulation (e.g., “success”, “error”, or a summary string). | string |
