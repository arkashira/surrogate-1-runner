Frame processing pipeline with optional brightness smoothing.

The baseline pipeline processes each frame independently.  The new
BrightnessSmoother class implements a simple moving-average style
smoothing that limits the change in brightness between consecutive
frames to a maximum of 10 %.  The smoother is applied after the
normalization step but before any downstream processing.

The implementation is deliberately lightweight to keep the
processing time increase below 5 %.  It operates on NumPy arrays
representing RGB frames and uses a running average of the mean
channel intensity as the brightness metric.
"""

import numpy as np
from typing import Optional, Iterable

# --------------------------------------------------------------------------- #
# Helper functions
# --------------------------------------------------------------------------- #

def frame_brightness(frame: np.ndarray) -> float:
    """
    Compute the mean brightness of an RGB frame.

    Parameters
    ----------
    frame : np.ndarray
        Frame data in shape (H, W, 3) with values in [0, 255].

    Returns
    -------
    float
        Mean brightness in the same range as the input values.
    """
    return float(frame.mean())

    Returns
    -------
    np.ndarray
        Brightness-adjusted frame clipped to [0, 255].
