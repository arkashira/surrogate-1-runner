### Prerequisites
Before starting, ensure you have:
- [ ] Git installed (`git --version`)
- [ ] Python 3.10+ (`python3 --version`)
- [ ] Node.js 18+ (`node --version`) *(Optional for advanced features)*
- [ ] Docker (optional, for containerized runs)

### Step 1: Clone the Repository
```bash
git clone https://github.com/axentx/surrogate-1.git
cd surrogate-1
```

### Step 2: Install Dependencies
```bash
# Create and activate virtual environment
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate

# Install Python dependencies
pip install -r requirements.txt

# Install Node dependencies (if needed)
npm install
```

### Step 3: Configure Environment
Create a `.env` file in the project root with these required keys:
```env
# Required: HuggingFace API token (get from https://huggingface.co/settings/tokens)
HF_TOKEN="hf_xxxxxxxxxxxxx"

# Required: Dataset repository
HF_DATASET_REPO="axentx/surrogate-1-training-pairs"

# Optional: Number of parallel workers (default 16)
WORKER_COUNT=16

# Optional: Output directory
OUTPUT_DIR="./outputs"

# Optional: Log level (DEBUG, INFO, WARN, ERROR)
LOG_LEVEL="INFO"
```

### Step 4: Initialize Workspace
```bash
# Generate shard assignments
./bin/dataset-enrich.sh --init

# Validate configuration
./bin/dataset-enrich.sh --validate
```

### Step 5: Run Ingestion Pipeline
```bash
# Test run with single worker
./bin/run-worker.sh --shard 0

# Full parallel run (16 workers)
./bin/run-workers.sh --parallel 16
```

### Configuration Reference
| Setting          | Description                     | Default/Example                     |

| `LOG_LEVEL`      | Logging verbosity                | `"INFO"`                            |

### Troubleshooting
| Issue                          | Solution                                                                 |
|--------------------------------|--------------------------------------------------------------------------|
| `ModuleNotFoundError`          | `pip install python-dotenv`                                             |
| Workers exit immediately       | Verify `.env` is correctly formatted and all required keys are present |
| Invalid token                 | Ensure token has write permissions and is correctly pasted in `.env`    |
