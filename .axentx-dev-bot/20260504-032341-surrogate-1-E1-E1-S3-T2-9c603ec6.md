## 1. Acceptance Criteria
- **AC1** – When a CGEvent `Cmd+V` paste fails, the resolver invokes the AppleScript fallback **within 60 seconds** of the failure detection.  
- **AC2** – If the AppleScript execution succeeds, the original paste operation is considered resolved and no further retries are performed.  
- **AC3** – If the AppleScript execution throws an error, the error is passed to `errorHandler.handle()` and **no retry** of the AppleScript is attempted.  
- **AC4** – For non‑AppleScript failures (e.g., network call, file I/O), the resolver retries the operation **exactly once** before delegating to the error handler.  
- **AC5** – `errorHandler.handle()` creates the log directory automatically when missing, writes **both** a plain‑text line and a JSON object to the same log file, and never propagates an exception to the caller.  
- **AC6** – When a global `__metrics__` object exists, a metric `resolver.appleScriptFailure` (or success) is emitted with appropriate tags.  
- **AC7** – All logging and metric emission failures are swallowed and recorded internally without crashing the application.
