## Final Synthesized Answer

### Diagnosis (integrated)
- **Primary failure**: Training still performs authenticated HF API calls (`list_repo_tree`, `load_dataset`) during data loading, causing 429s under iteration.
- **Missing persistence**: No file-list cache; each run re-enumerates folders via API instead of using a static manifest.
- **CDN bypass not used**: Data loader does not use CDN-only fetches (`resolve/main/...`) which bypass auth and rate limits.
- **Lightning Studio fragility**: Studio reuse is fragile; no pre-run status check → idle stop kills training and wastes quota.
- **Schema hygiene**: Enriched files carry extra metadata columns instead of strict `{prompt, response}` projection before/during upload.

### Proposed Change (integrated)
- Add/update `/opt/axentx/surrogate-1/train.py` to:
  - Accept `--file-list` JSON path produced once on the Mac orchestrator.
  - Use CDN-only fetches (`hf_hub_download` with CDN fallback) and **never** use `load_dataset`/`list_repo_*` during training.
  - Project each file to `{prompt, response}` at parse time.
  - Check Lightning Studio status and reuse or restart cleanly.
- Add `/opt/axentx/surrogate-1/scripts/build_file_list.py` to produce `file_list.json` for a given date folder (run once per date folder after rate-limit window clears).

```python
#!/usr/bin/env python3
"""
Usage (Mac orchestrator, run once per date folder after rate-limit window clears):
  HF_TOKEN=hf_xxx python scripts/build_file_list.py \
    --repo "axentx/surrogate-1-data" \
    --date-folder "batches/mirror-merged/2026-05-02" \
    --out "file_list.json"
"""
import argparse
import json
import os
import time

from huggingface_hub import HfApi

HF_TOKEN = os.getenv("HF_TOKEN")

def main() -> None:
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True)
    parser.add_argument("--date-folder", required=True)
    parser.add_argument("--out", required=True)
    args = parser.parse_args()

    api = HfApi(token=HF_TOKEN)

    # Non-recursive per-folder to minimize API calls
    entries = api.list_repo_tree(
        repo_id=args.repo,
        path=args.date_folder,
        repo_type="dataset",
        recursive=False,
    )

    files = [e.path for e in entries if e.type == "file"]
    # If nested folders exist, recurse one level only (bounded)
    for e in entries:
        if e.type == "folder":
            nested = api.list_repo_tree(
                repo_id=args.repo,
                path=e.path,
                repo_type="dataset",
                recursive=False,
            )
            files.extend(n.path for n in nested if n.type == "file")

    manifest = {
        "repo": args.repo,
        "date_folder": args.date_folder,
        "files": sorted(set(files)),
        "created_utc": int(time.time()),
        "note": "CDN-only; use resolve/main/ URLs during training to bypass HF API auth/rate-limits",
    }

    os.makedirs(os.path.dirname(os.path.abspath(args.out)) or ".", exist_ok=True)
    with open(args.out, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Wrote {len(files)} files to {args.out}")

if __name__ == "__main__":
    main()
```
