# Notification Configuration

This document describes the configuration options for error notifications in the surrogate-1 project.

## Error Notifications

Error notifications can be configured for different error types. The configuration is a map where the keys are error types and the values are the notification settings.

```yaml
error_notifications:
  critical:
    enabled: true
    recipients:
      - platform_engineer@example.com
  non_critical:
    enabled: false
    recipients: []
```

- `enabled`: A boolean indicating whether notifications are enabled for this error type.
- `recipients`: A list of email addresses to send notifications to.
```
