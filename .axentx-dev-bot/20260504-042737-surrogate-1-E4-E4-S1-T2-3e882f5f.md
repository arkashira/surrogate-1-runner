button that the user can click to start a session.

The module is intentionally lightweight and can be used both as a library
and as a small CLI tool for testing.

    """
    Retrieve the Slack OAuth token for a given user.

    `USER_SLACK_TOKEN_<USER_ID>`. This keeps the token out of source control
    and allows per-user authentication.

    Args:
        user_id: The unique identifier for the user.

    Returns:
        The Slack OAuth token as a string.

    Raises:
        KeyError: If the environment variable is not set.
    """
    env_var = f"USER_SLACK_TOKEN_{user_id}"
    token = os.getenv(env_var)
    if not token:

    return token
