## Final Consolidated Solution

### 1. Diagnosis (merged)
- **No persisted, date-scoped manifest**: training scripts re-enumerate HF repos via authenticated API on every run, exposing 429 rate-limit risk and non-deterministic inputs.
- **Non-CDN data path**: training likely performs authenticated fetches during data loading instead of using pre-listed CDN URLs.
- **Schema hazards**: `load_dataset(streaming=True)` on heterogeneous repos triggers `pyarrow.CastError` on mixed-schema files.
- **Lightning Studio quota burn**: reuse is not enforced; new `Studio(create_ok=True)` calls spin up fresh instances and burn quota.
- **Local heavy lifting**: Mac may run `model.from_pretrained()` or heavy ingestion locally instead of offloading to Lightning Studio / HF Space / Cerebras.

### 2. Proposed Change (merged + corrected)
Create a deterministic, CDN-only ingestion + training contract with strict reuse and schema enforcement:

- **Generate manifest once per date** on Mac (or any dev box) with a single `list_repo_tree` call; persist `manifest-{date}.json` containing `{file_path, cdn_url, sha256, size}`.
- **Train with zero HF API calls**: accept `--manifest` and stream files via CDN URLs using `datasets` with `streaming=True` and explicit schema projection.
- **Enforce Studio reuse**: list running studios and reuse; only start new if none running and quota allows.
- **Project and upload**: download via CDN, project strictly to `{prompt, response}`, write to `batches/mirror-merged/{date}/{slug}.parquet`, upload to HF dataset repo.
- **Schema guard**: validate and coerce types before parquet write to prevent `CastError`.

```python
#!/usr/bin/env python3
"""
Generate a CDN-only manifest for one date folder.
Run on Mac (or any dev box) when rate-limit window is clear.
"""
import json, os, sys, datetime
from huggingface_hub import list_repo_tree

REPO = os.getenv("HF_DATASET_REPO", "axentx/surrogate-1-data")
DATE = os.getenv("INGEST_DATE", datetime.date.today().isoformat())
OUT = os.getenv("MANIFEST_OUT", f"manifest-{DATE}.json")

def build_manifest() -> None:
    entries = []
    try:
        tree = list_repo_tree(repo_id=REPO, path=DATE, recursive=True)
    except Exception as e:
        print(f"list_repo_tree failed: {e}", file=sys.stderr)
        sys.exit(1)

    for node in tree:
        if node.type != "file":
            continue
        cdn_url = (
            f"https://huggingface.co/datasets/{REPO}/resolve/main/"
            f"{DATE}/{node.path}"
        )
        entries.append({
            "file_path": f"{DATE}/{node.path}",
            "cdn_url": cdn_url,
            "size": getattr(node, "size", None),
        })

    manifest = {
        "repo": REPO,
        "date": DATE,
        "generated_at": datetime.datetime.utcnow().isoformat() + "Z",
        "entries": entries,
    }

    with open(OUT, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Wrote {len(entries)} entries to {OUT}")

if __name__ == "__main__":
    build_manifest()
```
