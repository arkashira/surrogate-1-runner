## Replay

Reconstruct a run from a line-delimited event log and produce a `RunSummary`.

- CLI: `node dist/events/replay.js <logPath>` — exits 0 and prints a JSON `RunSummary` to stdout for valid logs; exits non-zero and writes an error to stderr for invalid paths or malformed lines.
- Module: `import { ReplayEngine, reconstructRun }` — `reconstructRun(lines)` returns a deterministic `RunSummary` with per-symbol and total fill/qty/notional, sorted events, and error details.
- Deterministic & idempotent: events are sorted by timestamp ascending; repeated runs on the same log produce byte-for-byte identical summaries.
- Error handling: missing files throw with code `ENOENT`; malformed JSON lines include the line number; rejected/cancelled events are counted and reported in `errors[]` with `code` and `message`.
- Summary completeness: `RunSummary` includes `startedAt`, `endedAt`, per-symbol fill/qty/notional/fills, totals that reconcile with raw events, and per-error details.
