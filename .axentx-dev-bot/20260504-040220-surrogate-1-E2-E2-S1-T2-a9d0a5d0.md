* The New_Product module installed

To install the New_Product module, run the following command:
```bash
terraform init
terraform apply
```

To configure New_Product with Terraform, add the following code to your Terraform configuration file:
```terraform
module "new_product" {
  source = file("./new_product")

  # Configure New_Product settings
  new_product_settings = {
    # Add your New_Product settings here
  }
}
```
