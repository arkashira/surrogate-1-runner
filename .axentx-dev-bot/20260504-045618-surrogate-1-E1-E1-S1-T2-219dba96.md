| **`Set`‑style API (`completedSteps`, `skippedSteps`)** (Candidate 1) | Lets the UI reason about “done” vs “skipped” steps without worrying about duplicate entries. |

 *   currentIndex: number,    // index into `steps` of the step currently shown
 *   lastSeen: number         // timestamp of the last mutation
 * }
 *
 * All public functions work on this shape, converting the array fields to
 * `Set` objects only when returning values to callers.
 */

    if (!raw) return _emptyState();
    const parsed = JSON.parse(raw);
    return {
      steps: Array.isArray(parsed.steps) ? parsed.steps : [],
      completedSteps: Array.isArray(parsed.completedSteps) ? parsed.completedSteps : [],
