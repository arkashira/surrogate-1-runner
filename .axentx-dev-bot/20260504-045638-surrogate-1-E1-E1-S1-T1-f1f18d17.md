  - For a typical SDK response, the function extracts the following fields for each line item: `service`, `usageType`, `operation`, `amount`, `start`, `end`.  
  - Missing optional fields (e.g., `operation`) are handled gracefully (e.g., set to `undefined` or an empty string).
