- **A2**: `match(criteria, minScore)` filters out any `MatchResult` with `score < minScore`.

- **A4**: After the cache TTL expires, a subsequent `match` call recomputes results and returns a new array instance.

- **A6**: `notifyNewProvider(provider)` triggers all callbacks whose stored criteria match the provider; callbacks are invoked **once** per provider and do **not** throw errors even if the provider does not match any subscription.
