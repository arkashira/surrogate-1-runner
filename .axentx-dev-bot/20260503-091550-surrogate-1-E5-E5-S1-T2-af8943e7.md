1. **Acceptance criteria**
- GET /teams/:id/dashboard returns 200 and JSON with numeric fields: wauRequests (≥0), openRequests (≥0), slaBreachCountLast7Days (≥0), slaBreachCountLast30Days (≥0).
- Response excludes internal-only fields (e.g., internalNotes, internalFlags) for all returned request summaries.
- SLA breach count for N days includes only requests where current_status is not terminal AND status_updated_at + sla_target_hours < now() AND created_at within last N days.
- WAU requests count includes non-internal requests (is_internal = false) with activity (any update or event) within the last 7 days scoped to the requested team.
- Open requests count includes non-internal requests where current_status is not terminal and belongs to the requested team.
- Team scoping enforced: requests for other teams are never included in any count.
- Invalid team ID or malformed request returns 400; unauthorized requests return 401/403; server errors return 5xx with no stack trace leakage.
