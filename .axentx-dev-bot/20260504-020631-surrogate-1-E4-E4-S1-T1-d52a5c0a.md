# surrogate-1-runner

Parallel public-dataset ingest workers for the [axentx/surrogate-1-training-pairs](https://huggingface.co/datasets/axentx/surrogate-1-training-pairs) HuggingFace dataset.

## What This Does

Every 30 minutes (or on `workflow_dispatch`), GitHub Actions launches **16 parallel runners**. Each runner takes a deterministic 1/16 slice (`slug-hash bucket = SHARD_ID`) of the public dataset list defined in `bin/dataset-enrich.sh`, streams, normalizes per-schema, dedups via the central md5 hash store, and uploads its output to a unique path on the dataset repo.

## MapView and Overlay

### MapView

The `MapView` component renders interactive maps. It supports various map providers and allows customization of map styles.

**Basic Usage:**

```javascript
import { MapView } from 'surrogate-1';

function App() {
  return (
    <MapView
      center={{ lat: 51.505, lng: -0.09 }}
      zoom={13}
      mapProvider="google"
    />
  );
}
```

### Overlay

The `Overlay` component displays additional information on top of the map, supporting markers, polygons, and polylines.

**Basic Usage:**

```javascript
import { MapView, Overlay } from 'surrogate-1';

function App() {
  const markers = [
    { position: { lat: 51.5, lng: -0.09 }, label: "Marker 1" },
    { position: { lat: 51.51, lng: -0.1 }, label: "Marker 2" }
  ];

  return (
    <MapView
      center={{ lat: 51.505, lng: -0.09 }}
      zoom={13}
      mapProvider="google"
    >
      <Overlay markers={markers} />
    </MapView>
  );
}
```
