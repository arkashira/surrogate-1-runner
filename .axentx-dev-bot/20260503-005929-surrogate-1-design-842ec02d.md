# Final Design Decision: CDN-Bypass Ingestion with Pre-Enumerated Manifest

## Summary
Replace all `load_dataset`/HF-API data loading paths with **CDN-only ingestion** driven by a **pre-computed file manifest** generated on the Mac orchestrator.  
This eliminates HF API 429/1000-req and 128-commit rate limits, removes per-worker pagination, and fixes schema heterogeneity by projecting to `{prompt,response}` at parse time.

---

## Why this now (highest-value, <2h)
- **Correctness**: Removes primary runtime failure mode (HF API 429 during training epochs).
- **Actionability**: Single manifest generation + small train.py patch; no infra changes.
- **Constraints**: Fits Lightning/HF free-tier and keeps Mac as orchestrator-only.

---

## Concrete implementation plan (<2h)

### 1) Mac orchestrator: generate manifest once per date folder
Create `bin/gen-manifest.sh` (runs on Mac):
- Uses `huggingface_hub` to call `list_repo_tree(..., recursive=True)` once per `public-merged/<YYYY-MM-DD>`.
- Emits `data/file_manifest_<YYYY-MM-DD>.json` (array of `{"repo","path","size","lfs"}`).
- Commits/pushes to repo **or** uploads as workflow artifact for runners.

```bash
#!/usr/bin/env bash
set -euo pipefail
REPO="axentx/surrogate-1-training-pairs"
DATE="${1:-$(date +%Y-%m-%d)}"
OUT="data/file_manifest_${DATE}.json"

python3 - "$REPO" "$DATE" "$OUT" <<'PY'
import json, os, sys
from huggingface_hub import list_repo_tree

repo, date, out = sys.argv[1], sys.argv[2], sys.argv[3]
path = f"public-merged/{date}"
items = list_repo_tree(repo=repo, path=path, recursive=True)
files = [
    {"repo": repo, "path": i.path, "size": i.size, "lfs": getattr(i, "lfs", None)}
    for i in items if i.type == "file"
]
with open(out, "w") as f:
    json.dump(files, f, indent=2)
print(f"Wrote {len(files)} files to {out}")
PY
```

### 2) Update `bin/dataset-enrich.sh` to use CDN bypass
- Accept `FILE_MANIFEST` env var (path to manifest). If absent, fallback to legacy `load_dataset` with warning.
- Deterministic shard assignment: `hash(slug) % 16 == SHARD_ID`.
- Download via raw CDN URL (no Authorization header):
  `https://huggingface.co/datasets/${repo}/resolve/main/${path}`
- Project to `{prompt,response}` at parse time; write `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`.
- Single commit per shard via `huggingface_hub` upload.

set -euo pipefail

HF_TOKEN="${HF_TOKEN:-}"
REPO="axentx/surrogate-1-training-pairs"
DATE="${DATE:-$(date +%Y-%m-%d)}"
SHARD_ID="${SHARD_ID:-0}"
FILE_MANIFEST="${FILE_MANIFEST:-}"
OUT_DIR="batches/public-merged/${DATE}"
TS=$(date +%H%M%S)
OUT_FILE="${OUT_DIR}/shard${SHARD_ID}-${TS}.jsonl"

mkdir -p "$OUT_DIR"

shard_for() {
  local slug="$1"
  local hash
  hash=$(echo -n "$slug" | sha256sum | tr -d ' -')
  echo $(( 0x${hash:0:8} % 16 ))
}

  jq -r '.[] | .path' "$FILE_MANIFEST" | while read -r relpath; do

    [[ $(shard_for "$slug") -eq "$SHARD_ID" ]] || continue

    url="https://huggingface.co/datasets/${REPO}/resolve/main/${relpath}"
    tmp=$(mktemp)
    if curl -fsSL "$url" -o "$tmp"; then

import json, pyarrow.parquet as pq, sys
try:
  tbl = pq.read_table('$tmp')
  df = tbl.to_pandas()
  prompt_col = next((c for c in df.columns if 'prompt' in c.lower()), df.columns[0])
  response_col = next((c for c in df.columns if 'response' in c.lower() or 'completion' in c.lower()), df.columns[-1])
  for _, row in df.iterrows():
    out = {'prompt': str(row[prompt_col]), 'response': str(row[response_col])}
    print(json.dumps(out, ensure_ascii=False))
except Exception as e:
  sys.stderr.write(f'Parse error {e} for $relpath\\n')

      echo "Download failed: $url"
    fi
    rm -f "$tmp"
  done
else
