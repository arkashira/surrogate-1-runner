### 1) Pre-flight check (`bin/preflight.py`)

Single script run at the start of the workflow (or locally) that:
- Fetches HF `list_repo_tree` for today’s `public-merged/<date>/` once.
- Persists `runs/file-list.json` (shard → file mapping) and `runs/schema-probe.json` (first file’s projected `{prompt,response}` dtypes).
- Exits non-zero if repo unreachable, schema invalid, or no files for today.
- Emits `runs/preflight.json` with `{ok, date_folder, total_files, schema_ok, fetched_at}` for the workflow to gate the matrix.

def main(out_dir, owner, repo_name):

    api = HfApi()

    prefix = f"batches/public-merged/{date_folder}/"

    try:
        files = list(api.list_repo_tree(owner, repo_name, path_in_repo="batches/public-merged", recursive=True))

        sys.exit(1)

    # map shard index -> files (simple deterministic split)
    shard_count = 16
