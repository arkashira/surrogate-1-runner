## Quick Start

```python
from surrogate1 import Functor, Monad

# A simple Functor that wraps a value
class Box(Functor):
    def __init__(self, value):
        self.value = value

    def map(self, fn):
        return Box(fn(self.value))

# A simple Monad that supports flat_map (bind)
class Maybe(Monad):
    def __init__(self, value):
        self.value = value

    def bind(self, fn):
        if self.value is None:
            return self  # propagate None
        return fn(self.value)

# Using the abstractions
result = Box(10).map(lambda x: x * 2).map(str)
print(result.value)   # → "20"

maybe = Maybe(5).bind(lambda x: Maybe(x * 3))
print(maybe.value)    # → 15
```

## Functor Pattern

A **Functor** is a container that can be *mapped* over.  
Only the `map` method is required.

### Example – Transforming a list of records

```python
from surrogate1 import Functor

class ListFunctor(Functor):
    def __init__(self, items):
        self.items = items

    def map(self, fn):
        return ListFunctor([fn(item) for item in self.items])

# Sample data
records = ListFunctor([
    {"id": 1, "text": "hello"},
    {"id": 2, "text": "world"},
])

# Upper‑case every `text` field
uppercased = records.map(lambda r: {**r, "text": r["text"].upper()})
print(uppercased.items)
# → [{'id': 1, 'text': 'HELLO'}, {'id': 2, 'text': 'WORLD'}]
```

## Monad Pattern

A **Monad** adds the ability to *flatten* nested contexts via `bind` (also called `flat_map`).  
The only law you need to respect is that `bind` must return a monadic value.

### Example – Safe division with `Maybe`

```python
from surrogate1 import Monad

class Maybe(Monad):
    def __init__(self, value):
        self.value = value

    def bind(self, fn):
        if self.value is None:
            return self
        return fn(self.value)

def safe_div(x, y):
    return Maybe(x / y) if y != 0 else Maybe(None)

result = Maybe(10).bind(lambda x: safe_div(x, 2)).bind(lambda x: safe_div(x, 0))
print(result.value)   # → None (division by zero was safely propagated)
```

## Tutorial – Common Use Cases

Below is a step‑by‑step walkthrough that combines both patterns to process the public dataset used by **surrogate-1‑runner**.

1. **Load raw records** – wrapped in a `ListFunctor`.
2. **Normalize** – map a normalisation function over each record.
3. **Validate** – bind a `Maybe`‑based validator that can short‑circuit on failure.
4. **Upload** – map the final upload step.

```python
from surrogate1 import Functor, Monad

# 1️⃣ Load raw data
raw = ListFunctor([
    {"slug": "a1", "payload": "raw1"},
    {"slug": "b2", "payload": "raw2"},
])

# 2️⃣ Normalise each record
def normalise(rec):
    # pretend we do heavy schema work here
    rec["payload"] = rec["payload"].upper()
    return rec

norm = raw.map(normalise)

# 3️⃣ Validate – returns Maybe to abort on bad data
def validate(rec):
    return Maybe(rec) if len(rec["payload"]) > 0 else Maybe(None)

validated = ListFunctor([
    validate(r).value for r in norm.items
])  # we unwrap for simplicity in this tutorial
