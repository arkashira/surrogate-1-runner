## 1. Acceptance Criteria  
- **Dashboard Tab**: The `Usage` tab must display three metrics:  
  1. **Pipeline Count** – integer count of pipelines created in the last 30 days.  
  2. **Total Data Ingested** – sum of `dataSizeBytes` for all pipeline runs in the last 30 days, expressed in megabytes (MB) with two decimal places.  
  3. **Average Run Time** – average duration of pipeline runs in the last 30 days, expressed in seconds with one decimal place.  
- **Data Refresh**:  
  - On initial page load, the dashboard must request usage data via the `UsageService`.  
  - Subsequent updates must be received via WebSocket messages sent by `UsageWebSocketHandler`.  
- **Time Window**: All statistics must be calculated strictly for the period `[now - 30 days, now]`.  
- **WebSocket Interaction**:  
  - Sending a message with payload `{ "action": "refresh" }` must trigger an immediate calculation and broadcast of current statistics.  
  - The JSON payload sent over the socket must match the structure of `UsageStatistics` (`pipelineCount`, `totalDataIngestedMB`, `averageRunTimeSec`).  
- **Edge Cases**:  
  - If no pipelines or runs exist in the last 30 days, all metrics must be `0`.  
  - If a pipeline run has a `null` duration or data size, it must be excluded from averages and sums respectively.

  }

  @Test
  void shouldCalculateMetricsCorrectly() {
    Pipeline p1 = new Pipeline("id1", LocalDateTime.now().minusDays(5));
    Pipeline p2 = new Pipeline("id2", LocalDateTime.now().minusDays(10));
    when(pipelineRepo.findAllByCreatedAtAfter(any())).thenReturn(List.of(p1, p2));

    PipelineRun r1 = new PipelineRun("run1", p1, 500_000L, Duration.ofSeconds(30));
    PipelineRun r2 = new PipelineRun("run2", p2, 1_500_000L, Duration.ofSeconds(90));
    when(runRepo.findAllByCreatedAtAfter(any())).thenReturn(List.of(r1, r2));

    UsageStatistics stats = usageService.calculateUsage();

    assertEquals(2, stats.getPipelineCount());
    assertEquals(2.0, stats.getTotalDataIngestedMB(), 0.01); // 2 MB
    assertEquals(60.0, stats.getAverageRunTimeSec(), 0.1);   // (30+90)/2
  }

  @Test
  void shouldExcludeNullDurationsAndDataSizes() {
    PipelineRun r1 = new PipelineRun("run1", new Pipeline("id1", LocalDateTime.now()), null, Duration.ofSeconds(20));
    PipelineRun r2 = new PipelineRun("run2", new Pipeline("id2", LocalDateTime.now()), 1_000_000L, null);
    when(runRepo.findAllByCreatedAtAfter(any())).thenReturn(List.of(r1, r2));

    UsageStatistics stats = usageService.calculateUsage();

    assertEquals(0, stats.getPipelineCount()); // no pipelines counted
    assertEquals(1.0, stats.getTotalDataIngestedMB(), 0.01); // only r2 counted
    assertEquals(20.0, stats.getAverageRunTimeSec(), 0.1);   // only r1 counted
  }
