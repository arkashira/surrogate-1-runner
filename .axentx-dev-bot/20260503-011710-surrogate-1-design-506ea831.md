## Final consolidated implementation (best of both proposals)

**Decision**  
Replace recursive `list_repo_files` and per-file authenticated HF API calls with **one non-recursive `list_repo_tree` per folder + CDN-only fetches + strict `{prompt,response}` projection**.  
This eliminates HF API 429 risk, per-file auth overhead, pyarrow `CastError` from mixed schemas, and keeps ingestion within the 128-commit/hr budget.

---

## Implementation plan (≤2h)

1. **Audit and update entrypoint** (`bin/dataset-enrich.sh`)  
   - Accept a date-folder arg (e.g., `2026-05-03`) so each job processes one folder.  
   - On the orchestrator (once per folder), list files non-recursively and persist the list.  
   - Workers consume the persisted file list and fetch via CDN (no auth for raw files).

2. **Create `bin/list_folder_tree.py`**  
   - Non-recursive tree listing for a folder.  
   - Outputs JSON list of file paths to stdout (or file).  
   - Optional `HF_TOKEN` only for listing private repos; public repos need none.

3. **Update ingestion worker** (`bin/ingest_shard.py` or inline in `dataset-enrich.sh`)  
   - Read file list from JSON.  
   - Optionally shard by `hash(file) % 16 == SHARD_ID`.  
   - For each file:
     - Use `hf_hub_download` (or direct CDN URL) to fetch raw file.  
     - Read with `pyarrow.parquet.read_table(columns=["prompt","response"])` when possible.  
     - On missing columns or schema errors, project best-effort to `{prompt,response}` with nulls.  
     - Drop all other columns.  
   - Dedup by md5 of `prompt\0response` using existing `lib/dedup.py` or local set per shard.  
   - Stream rows into `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`.

4. **Use CDN URLs when possible**  
   - For public files: `https://huggingface.co/datasets/{repo}/resolve/main/{path}` (no auth).  
   - Fallback to `hf_hub_download` for private repos or when CDN fetch fails.

5. **Schema enforcement**  
   - Keep only `prompt`, `response`.  
   - If missing, fill with `null`.  
   - Do not add extra metadata columns (`source`, `ts`) to preserve surrogate-1 schema contract.

6. **Output and filenames**  
   - Preserve: `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`.

7. **Local test**  
   - Run on a small folder subset.  
   - Verify:
     - No 429 errors.  
     - Output JSONL contains only `prompt`, `response`.  
     - Dedup correctly marks duplicates.

8. **Deploy**  
   - Commit changes to `bin/`.  
   - Trigger `workflow_dispatch` to verify in CI.

### `bin/list_folder_tree.py`
```python
#!/usr/bin/env python3
"""
List non-recursive file tree for a HF dataset folder.
Usage:
  HF_TOKEN=... python list_folder_tree.py --repo axentx/surrogate-1-training-pairs --path batches/raw/2026-05-03
Outputs JSON list of file paths to stdout.
"""
import argparse
import json
import os
from huggingface_hub import HfApi

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True)
    parser.add_argument("--path", required=True)
    args = parser.parse_args()

    token = os.getenv("HF_TOKEN")
    api = HfApi(token=token)

    # non-recursive listing
    tree = api.list_repo_tree(
        repo_id=args.repo,
        path=args.path,
        repo_type="dataset",
        recursive=False
    )
    files = [item.path for item in tree if item.type == "file"]

    print(json.dumps(files, indent=2))

if __name__ == "__main__":
    main()
```
