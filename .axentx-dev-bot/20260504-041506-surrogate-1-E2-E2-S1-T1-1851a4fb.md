## 1️⃣  What we are building  

A **single source‑of‑truth repository** (`/opt/axentx/surrogate-1/templates`) that holds **all Surrogate‑1 ingestion‑pattern templates**.  

* Each template lives in its own folder (`templates/<kebab‑id>/`).  
* The folder contains a **manifest** (`manifest.json`) that describes the template and points to the concrete files (scripts, configs, etc.).  
* A **gallery index** (`index.json`) lives at the root and lists every template together with version‑wide metadata.  
* All files are validated against **two JSON‑Schema documents** – one for the gallery index and one for the per‑template manifest.  
* A minimal `.gitignore` and a starter `README.md` make the repo ready for collaborative development.

The design merges the **simplicity** of Candidate 1 (clear component model, required timestamps) with the **richness** of Candidate 2 (gallery‑wide versioning, tags, dependencies, reusable file‑type definitions).  

---

```
/opt/axentx/surrogate-1/templates/
│
├─ .gitignore                # never commit generated artefacts

```
