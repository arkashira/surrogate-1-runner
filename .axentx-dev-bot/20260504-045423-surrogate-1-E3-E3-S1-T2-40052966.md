- **AC-4**: All returned alternatives have `performance_score` ≥ the user's `min_performance` threshold
- **AC-5**: Alternatives are sorted by `performance_score` in descending order within each component's alternatives list
- **AC-6**: The endpoint returns HTTP 200 for valid requests and HTTP 422 for invalid input payloads
- **AC-7**: Empty alternatives list is returned when no viable alternatives exist for a given component
