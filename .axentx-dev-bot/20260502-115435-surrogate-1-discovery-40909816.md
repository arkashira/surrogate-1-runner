### 1. Diagnosis (Consolidated)
- **No persisted manifest**: re-enumerates HF repos on every run → rate-limit exposure and non-deterministic training inputs.
- **Authenticated API on hot path**: training uses `load_dataset(streaming=True)` or API calls during epochs → 429s and quota burn.
- **Schema hazards**: mixed-schema parquet/jsonl (extra `source`, `ts`, etc.) → `pyarrow.CastError` on late projection.
- **Lightning Studio churn**: no reuse guard → can burn 80hr/mo quota on idle/stop churn.
- **Mac role confusion**: Mac used for orchestration + experimentation → risk of accidental local `from_pretrained` and compute waste. Enforce Mac=CLI-only for training orchestration.

### 2. Proposed Change (Concrete)
- Add **one-time manifest generation** that lists a date folder and saves CDN-only paths.
- Update training to **load manifest, fetch files via CDN (or one-time `hf_hub_download` cache), project `{prompt,response}` immediately**, and stream to training without authenticated API calls during epochs.
- Add **Lightning Studio reuse guard** keyed by name.
- Enforce **Mac=CLI-only** boundary: no training imports or `from_pretrained` on Mac; Mac only runs manifest generation and submits training jobs.

```python
#!/usr/bin/env python3
"""
Generate a CDN-only manifest for a date folder.
Run from Mac (or any env with huggingface_hub) once per snapshot.
Mac=CLI-only: do not use this environment for training imports.
"""
import json
import os
import sys
from datetime import datetime
from pathlib import Path

from huggingface_hub import HfApi

REPO_ID = os.getenv("HF_DATASET_REPO", "axentx/surrogate-1-data")
DATE_FOLDER = os.getenv("DATE_FOLDER", datetime.utcnow().strftime("%Y-%m-%d"))
OUT_DIR = Path(__file__).parent.parent.parent / "manifests"
OUT_DIR.mkdir(exist_ok=True, parents=True)

def main() -> None:
    api = HfApi()
    files = api.list_repo_tree(
        repo_id=REPO_ID,
        path=DATE_FOLDER,
        repo_type="dataset",
        recursive=True,
    )

    entries = []
    for f in files:
        if not f.path.endswith((".jsonl", ".parquet", ".json")):
            continue
        # CDN URL bypasses API auth/rate limits
        cdn_url = (
            f"https://huggingface.co/datasets/{REPO_ID}/resolve/main/{f.path}"
        )
        entries.append(
            {
                "path": f.path,
                "cdn_url": cdn_url,
                "size": getattr(f, "size", None),
            }
        )

    out_path = OUT_DIR / f"manifest-{DATE_FOLDER}.json"
    out_path.write_text(json.dumps(entries, indent=2))
    print(f"Wrote {len(entries)} entries to {out_path}")

if __name__ == "__main__":
    main()
```
