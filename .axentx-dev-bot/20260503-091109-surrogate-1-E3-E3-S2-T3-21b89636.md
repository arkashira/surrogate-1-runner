## Key synthesis decisions

1. **Capture phase listener** (`true`): intercepts `/` before browser-level shortcuts (Firefox address bar, etc.) while still allowing normal typing in form fields.

2. **Precise editable detection**:
   - `isEditableTarget` combines Candidate 1’s thoroughness (textarea/select/contenteditable) with Candidate 2’s structure.
   - `isTextualInput` excludes non-textual input types (button, checkbox, etc.) so `/` doesn’t accidentally trigger when clicking buttons.

3. **Focus/blur behavior**:
   - `focusSearch` stores `lastNonSearchFocus` **only when search is actually focused** (not on every keydown).
   - `blurSearch` restores focus and clears the tracker to avoid stale references.
   - Cursor placed at end (not `.select()`) to avoid overwriting existing search text if any.

4. **Safety**:
   - All focus calls wrapped in try/catch.
   - Early returns when `searchInput` is missing.
   - Prevents default only when handling the shortcut (avoids interfering with normal `/` typing in form fields).
