## 1.  What we’re building

A **real‑time Azure cost‑dashboard** that

| Feature | Why it matters | How we’ll implement it |
|---------|----------------|------------------------|
| **Live cost metrics** | Spot expensive resources instantly | Query Azure Cost Management API every *N* minutes and cache results in Redis. |
| **Aggregated summaries** | Quick view of spend by service, type, tag | Sum metrics per resource type & service, compute trend vs. previous period. |
| **Audit trail** | Compliance & debugging | Persist every query & action in a PostgreSQL table (`audit_log`). |
| **Extensible to other clouds** | Future‑proof | Abstract the client behind a `CostProvider` interface. |
| **Secure credentials** | Never hard‑code secrets | Use Azure Managed Identities or Azure Key Vault. |
| **Observability** | Detect failures | Export Prometheus metrics (`cost_query_duration`, `cost_query_errors`). |
| **Deployment** | Easy CI/CD | Docker image, Helm chart, GitHub Actions. |

class Settings(BaseSettings):
    # Core

    # Azure
    subscription_id: str = Field(..., env="AZURE_SUBSCRIPTION_ID")
    tenant_id: str = Field(..., env="AZURE_TENANT_ID")

    client_secret: str = Field(..., env="AZURE_CLIENT_SECRET")
    # DB
    audit_db_url: str = Field(..., env="AUDIT_DB_URL")
    # Cache
    redis_url: str = Field(..., env="REDIS_URL")
    # Prometheus
    prometheus_port: int = Field(8001, env="PROMETHEUS_PORT")

settings = Settings()

### 3.2 `models.py`

```python
from datetime import datetime
from typing import Dict, List, Optional
from dataclasses import dataclass, asdict
from pydantic import BaseModel

# ---- Domain models ----
@dataclass
class CostMetric:
    timestamp: datetime
    resource_id: str
    resource_name: str
    resource_type: str
    cost: float
    currency: str = "USD"
    tags: Optional[Dict[str, str]] = None

    def to_dict(self):
        d = asdict(self)
        d["timestamp"] = self.timestamp.isoformat()
        return d

class CostSummary(BaseModel):
    period_start: datetime
    period_end: datetime
    total_cost: float
    currency: str
    by_resource_type: Dict[str, float]
    by_service: Dict[str, float]
    trend: float  # % change vs previous period

# ---- API schemas ----
class RealtimeRequest(BaseModel):
    hours: int = 24

class SummaryRequest(BaseModel):
    days: int = 30

class AuditEntry(BaseModel):
    timestamp: datetime
    action: str
    user: str
    resource: str
    details: Dict[str, str]
```
