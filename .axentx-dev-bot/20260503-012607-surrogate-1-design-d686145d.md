## Final, consolidated implementation (single source of truth)

**Core idea (both candidates agree):**  
Replace per-file authenticated HF Hub API calls with **one non-recursive `list_repo_tree` per date folder + deterministic shard routing + CDN-only fetches**. This eliminates 429s and removes auth overhead for reads.

**Chosen details (favoring correctness + actionability):**
- Use a **single pre-step** to list the date folder and produce `filelist.json` (not per-shard) to avoid redundant calls and race conditions.
- Deterministic shard assignment by `hash(path) % TOTAL_SHARDS` (not slug) because filenames are stable and already encode the slug/date.
- Download exclusively via **CDN** (`https://huggingface.co/datasets/.../resolve/main/...`) with no Authorization header.
- Keep per-shard idempotent output: `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`.
- Central md5 dedup store remains source of truth; workers remain stateless.
- Retry/backoff for CDN downloads (network/HTTP 429/5xx) separate from API ratelimit logic.
- GitHub Actions: one matrix job per shard (0–15); a single “list” step runs once and shares `filelist.json` via workspace artifact (fast, reliable).

### bin/list_date_folder.py
```python
#!/usr/bin/env python3
"""
List immediate children of a date folder in axentx/surrogate-1-training-pairs
(non-recursive) and write filelist.json for workers.
"""
import json
import os
import sys
from huggingface_hub import HfApi

REPO = "axentx/surrogate-1-training-pairs"
API = HfApi(token=os.getenv("HF_TOKEN"))

def main(date_folder: str, out_path: str = "filelist.json"):
    entries = API.list_repo_tree(repo_id=REPO, path=date_folder, recursive=False)
    files = [
        {"path": e.path, "sha": getattr(e, "commit_oid", None) or getattr(e, "sha", None), "size": getattr(e, "size", None)}
        for e in entries
        if not e.path.endswith("/")  # files only
    ]
    with open(out_path, "w") as f:
        json.dump(files, f, indent=2)
    print(f"Wrote {len(files)} entries to {out_path}")

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Usage: list_date_folder.py <date-folder> [out.json]")
        sys.exit(1)
    main(sys.argv[1], sys.argv[2] if len(sys.argv) > 2 else "filelist.json")
```
