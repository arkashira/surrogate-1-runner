- **AC1:** POST `/api/recordings/schedule` returns 400 if URL is not a valid HTTP/HTTPS URL format
- **AC2:** POST `/api/recordings/schedule` returns 400 if `start_time` is not in the future (>= now + 1 minute)
- **AC3:** POST `/api/recordings/schedule` returns 400 if `end_time` <= `start_time` or `end_time` - `start_time` > 24 hours

- **AC5:** Recording is persisted in database with all provided fields (url, start_time, end_time, status, s3_key=null)
- **AC6:** Background job triggers at `start_time` (±5 seconds) and stops at `end_time` (±5 seconds)
- **AC7:** Completed recording is uploaded to S3 as MP3 file, status updated to "completed", s3_key populated
