## Final Implementation Plan (≤2h)

Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:

- Accepts `SHARD_ID` and `SHARD_TOTAL` (16) from GitHub Actions matrix.
- Uses a pre-generated `file-list.json` (Mac-side, one `list_repo_tree` call per date folder) to avoid recursive HF API pagination and 429s.
- Deterministically assigns files via `hash(slug) % SHARD_TOTAL` (not positional modulo) to tolerate manifest ordering changes and keep shard assignments stable across reruns.
- Downloads only assigned shard files via HF CDN (`resolve/main/...`) — zero Authorization header, bypasses `/api/` rate limits.
- Projects heterogeneous files to `{prompt, response}` at parse time (avoids pyarrow CastError from mixed schemas).
- Deduplicates via central `lib/dedup.py` md5 store.
- Writes deterministic output: `batches/public-merged/{date}/shard{N}-{HHMMSS}.jsonl`.
- Reuses existing Lightning Studio pattern for any local dev (not needed here — workers run in GitHub Actions).
- Adds proper Bash shebang and executable bit for any wrapper scripts; sets `SHELL=/bin/bash` in cron if used.

    """
    Return {"prompt": ..., "response": ...} or None if unparseable.
    Keep this small and defensive to avoid schema CastErrors.
    """
    # Heuristic: look for common key names
    prompt_keys = {"prompt", "instruction", "input", "question", "user"}
    response_keys = {"response", "output", "answer", "assistant", "completion"}

    # Try direct keys first
    for pk in prompt_keys:
        if pk in raw:

    # Unparseable for our training pair
    return None


def download_via_cdn(repo: str, path: str, retries: int = 3, backoff: int = 5) -> bytes:
    url = CDN_DOWNLOAD_BASE.format(repo=repo, path=path)
    for attempt in range(1, retries + 1):
        try:
            resp = requests.get(url, timeout=30)
            resp.raise_for_status()
            return resp.content
        except Exception as exc:
            if attempt == retries:
                raise
            time.sleep(backoff * attempt)
    raise RuntimeError("unreachable")
