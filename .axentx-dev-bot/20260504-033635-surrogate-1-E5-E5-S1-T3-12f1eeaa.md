   ```

   ```

### Running the Framework

To run the orchestration framework, execute the following command:

```bash
python main.py
```

This will launch the framework and start processing the dataset.

### Configuration

The framework can be configured via a `config.yaml` file. Here is an example configuration:

```yaml
dataset_path: "/path/to/dataset"
output_path: "/path/to/output"
num_workers: 16
interval_minutes: 30
```

Place this file in the root directory of the project.
