## 1) Fix the rate-limit bottleneck (do this first)

**What to change**
- Replace any recursive `list_repo_files()` or repeated `load_dataset()` calls in `bin/dataset-enrich.sh` with a **single `list_repo_tree`** for the target date folder.
- Emit a **file manifest** (`file-manifest-<date>.json`) that lists every file path + size + optional checksum.
- Keep all shard outputs and summaries in `batches/public-merged/<date>/` so the manifest is stable and CDN-only resolvable.

**Why this is correct**
- One tree call per run avoids 429s and wastes no quota.
- A manifest lets every downstream stage (including training) use raw CDN URLs exclusively, which removes API usage during data streaming.

**Concrete code**

Add to `bin/dataset-enrich.sh` near the top (after date is determined):

```bash
# -- produce file manifest once per run --
DATE=$(date -u +%Y-%m-%d)
MANIFEST="file-manifest-${DATE}.json"
MANIFEST_PATH="batches/public-merged/${DATE}/${MANIFEST}"

# list_repo_tree via huggingface_hub (install if needed)
python3 -c "
import json, os
from huggingface_hub import HfApi
api = HfApi()
repo_id = 'axentx/surrogate-1-training-pairs'
prefix = 'batches/public-merged/${DATE}/'
entries = []
for f in api.list_repo_tree(repo_id=repo_id, path=prefix, repo_type='dataset'):
    if f.path.endswith('/'):  # skip directory placeholders
        continue
    entries.append({
        'path': f.path,
        'size': getattr(f, 'size', None),
        'lfs': getattr(f, 'lfs', None) is not None
    })
out = {'date': '${DATE}', 'generated_at': '${STARTED_AT:-$(date -u +%Y-%m-%dT%H:%M:%SZ)}', 'entries': entries}
import sys; json.dump(out, sys.stdout)
```

**Training-side usage (optional, 10 min)**

Create `lib/cdn_loader.py`:

```python
import json, urllib.request, os
from typing import List, Dict

HF_REPO = "axentx/surrogate-1-training-pairs"

def load_manifest(date: str) -> List[Dict]:
    url = f"https://huggingface.co/datasets/{HF_REPO}/resolve/main/batches/public-merged/{date}/file-manifest-{date}.json"
    with urllib.request.urlopen(url) as r:
        data = json.load(r)
    return data["entries"]

def cdn_url(path: str) -> str:
    return f"https://huggingface.co/datasets/{HF_REPO}/resolve/main/{path}"
```

Training script imports `load_manifest(date)` once, iterates `cdn_url(e["path"])`, and streams with `urllib`/`requests`/`aiohttp`—zero HF API calls during epoch loops.
