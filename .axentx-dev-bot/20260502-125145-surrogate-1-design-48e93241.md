### 1. Diagnosis (Consolidated)
- **No persisted, date-scoped manifest**: every training run re-enumerates HF repos via authenticated API → risk of 429 rate-limits and non-deterministic inputs.
- **Authenticated data loading during training**: `load_dataset`/`list_repo_files` consume limited API quota and expose training to auth/rate failures.
- **No deterministic snapshot**: reproducibility and auditability gaps.
- **Lightning Studio reuse/idle-stop not codified**: risk of quota waste and lost progress.
- **Schema hygiene weak**: raw enriched files may carry extra columns (`source`, `ts`) instead of strict `{prompt, response}` projection before training.

---

### 2. Proposed Change (Single Artifact)
Create `/opt/axentx/surrogate-1/train_manifest.py` that:

1. **Orchestration host (Mac/CI)** lists one date-scoped folder via HF API **once**, saves deterministic `manifests/{date}/files.json` (path + size + SHA).
2. **Training side** loads only those files via **CDN-only URLs** (`https://huggingface.co/datasets/.../resolve/main/...`) with **zero authenticated calls** during training.
3. **Lightning Studio reuse + idle-stop guard** codified to prevent quota waste and lost progress.
4. **Strict schema projection** on read: keep only `{prompt, response}` and drop nulls.

---

#!/usr/bin/env python3
"""
Generate and use a CDN-only manifest for HF dataset training.
- Host side (Mac/CI): list once -> manifests/{date}/files.json
- Training side (Lightning): consume manifest and fetch via CDN (no HF API auth during training).
"""
import json, os, sys, io, hashlib
from datetime import datetime
from pathlib import Path

try:
    from huggingface_hub import HfApi
except ImportError:
    import subprocess
    subprocess.check_call([sys.executable, "-m", "pip", "install", "huggingface-hub"])
    from huggingface_hub import HfApi

HF_REPO = os.getenv("HF_DATASET_REPO", "datasets/axentx/surrogate-1-enriched")
MANIFEST_ROOT = Path(__file__).parent / "manifests"
DATE_FOLDER = os.getenv("TRAIN_DATE", datetime.utcnow().strftime("%Y-%m-%d"))

# ---- Manifest builder ----
def build_manifest() -> Path:
    api = HfApi()
    out_dir = MANIFEST_ROOT / DATE_FOLDER
    out_dir.mkdir(parents=True, exist_ok=True)
    manifest_path = out_dir / "files.json"

    entries = api.list_repo_tree(repo_id=HF_REPO, path=DATE_FOLDER, recursive=False)
    files = []
    for e in entries:
        if e.type != "file":
            continue
        if not e.path.endswith(".parquet"):
            continue
        files.append({
            "path": e.path,
            "size": e.size or 0,
            "sha": e.lfs.get("oid", None) if getattr(e, "lfs", None) else None
        })

    payload = {
        "date": DATE_FOLDER,
        "repo": HF_REPO,
        "generated_at": datetime.utcnow().isoformat() + "Z",
        "files": files
    }

    with open(manifest_path, "w") as f:
        json.dump(payload, f, indent=2)
    print(f"Manifest written: {manifest_path}")
    return manifest_path
