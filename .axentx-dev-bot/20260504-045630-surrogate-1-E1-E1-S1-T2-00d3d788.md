## Unified Onboarding Flow for **Surrogate‑1**

Below is a single, production‑ready React component that merges the strongest ideas from the two proposals:

| What we kept | Why |
|--------------|-----|
| **Step data in an array** (title, content, optional `cta`) | Simple, declarative, easy to extend. |
| **`onFinish` callback prop** | Lets the parent launch the first ingestion run (or any other action) without hard‑coding side‑effects inside the component. |
| **Progress bar + dot indicator** | Gives users a clear visual cue of where they are (both a bar and dots work well together). |
| **Previous / Next navigation** | Allows users to go back – a usability win over the “only Next” version. |
| **Final “Start First Run” CTA** (styled differently) | Highlights the important action at the end of the flow. |
| **Responsive, accessible styling** (focus states, disabled button handling) | Improves keyboard & screen‑reader experience. |
| **Self‑contained CSS** (no external UI library) | Keeps the component portable and easy to ship with the repo. |

The resulting component is **fully functional**, **type‑safe** (via JSDoc), and **actionable** – just drop the files into `src/components/` and import `<OnboardingFlow onFinish={…} />` wherever you need the onboarding experience.

import React, { useState } from "react";

 * OnboardingFlow

 * optionally define a `cta` label; when present the button triggers `onFinish`
 * instead of moving to the next step.

export default function OnboardingFlow({ onFinish }) {
  // -------------------------------------------------------------------------
  // 1️⃣  Step definitions – edit/extend here
  // -------------------------------------------------------------------------

    {
      title: "Welcome to Surrogate‑1",
      content:
        "Surrogate‑1 helps you ingest and process public datasets at scale. " +
        "This short onboarding will get you up and running in under 10 minutes.",
    },
    {
      title: "Quick Tour",
      content:
        "• Parallel runners run every 30 min\n" +
        "• Each runner processes a deterministic slice of the dataset\n" +
        "• Results are automatically uploaded to the HuggingFace hub",
    },
    {
      title: "Connect Your Data",
      content:
        "Link your data sources (CSV, JSON, Parquet, …) and let Surrogate‑1 handle the rest.",
    },
    {
      title: "Configure Processing",
      content:
        "Set up your processing parameters with our intuitive UI – no code required.",
    },
    {
      title: "Your First Action",
      content:
        "Launch your first ingestion run now. Click the button below to start processing.",
      cta: "Start First Run",
    },
  ];

  // -------------------------------------------------------------------------
  // 2️⃣  Local UI state
  // -------------------------------------------------------------------------
  const [current, setCurrent] = useState(0);
  const [completed, setCompleted] = useState(false);

  const isLastStep = current === steps.length - 1;
  const { title, content, cta } = steps[current];

  // -------------------------------------------------------------------------
  // 3️⃣  Handlers
  // -------------------------------------------------------------------------
  const goNext = () => {
    if (!isLastStep) setCurrent(current + 1);
    else setCompleted(true);
  };

  const goPrev = () => {
    if (current > 0) setCurrent(current - 1);
  };

  const handleCta = () => {
    if (onFinish) onFinish();
    // Reset flow so the user can start another run if they wish
    setCurrent(0);
    setCompleted(false);
  };
