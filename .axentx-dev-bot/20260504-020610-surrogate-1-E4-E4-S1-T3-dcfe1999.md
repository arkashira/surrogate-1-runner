# Custom Rule Plugins for Surrogate-1

This document explains how to write custom detection rules that can be loaded by the Surrogate-1 scanner. Custom rules allow security teams to enforce project-specific policies without modifying the core codebase.

## 1. Overview

Surrogate-1 supports two approaches for custom rules:

- **Class-based (recommended)**: Define a `Rule` class inheriting from `BaseRule` for more complex rules with metadata
- **Function-based**: Define a simple `detect` function for lightweight checks

Both approaches are automatically discovered—no manual registration required.

## 2. Directory Structure

Place custom rules in the `plugins/` directory at the repository root:

```
surrogate-1/
├── plugins/
│   ├── __init__.py
│   ├── my_custom_rule.py
│   └── no_trailing_whitespace.py
└── docs/
    └── custom_rules.md
```

The `__init__.py` can be empty; it marks the directory as a Python package.

## 3. Class-Based Approach (Recommended)

### BaseRule API

```python
from surrogate_1.rules.base import BaseRule, Detection
from typing import List

class BaseRule:
    """Base class for all custom rules."""
    name: str          # Human-readable rule name
    description: str   # Short description
    severity: str      # One of: "low", "medium", "high", "critical"

    def run(self, context: ScanContext) -> List[Detection]:
        """Execute the rule against the provided ScanContext."""
        ...
```

### Detection Object

```python
class Detection:
    """Represents a single finding."""
    rule_name: str
    message: str
    severity: str
    location: str   # e.g., file path or URL
    metadata: dict  # Optional additional info
```

### Example: Custom Rule

```python
# plugins/my_custom_rule.py
from surrogate_1.rules.base import BaseRule, Detection

class MyCustomRule(BaseRule):
    name = "My Custom Rule"
    description = "Detects usage of deprecated API X."
    severity = "high"

    def run(self, context):
        findings = []
        for file_path, content in context.files.items():
            if "deprecated_api_x" in content:
                findings.append(
                    Detection(
                        rule_name=self.name,
                        message="Deprecated API X used.",
                        severity=self.severity,
                        location=file_path,
                        metadata={"line": content.index("deprecated_api_x") + 1}
                    )
                )
        return findings
```
