package com.axentx.surrogate1.model;

import jakarta.validation.constraints.NotBlank;
import lombok.AllArgsConstructor;
import lombok.Getter;
import lombok.NoArgsConstructor;
import lombok.Setter;

/**
 * Immutable DTO that represents an API request that has drifted.
 */
@Getter
@Setter
@NoArgsConstructor
@AllArgsConstructor
public class ApiRequest {

    @NotBlank

    private String details;   // optional human‑readable description
}
```

*Why?*  
- `@NotBlank` guarantees that a signature is always present.
