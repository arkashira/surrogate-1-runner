## Final Implementation Plan (≤2h)

Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:

- Accepts `SHARD_ID`, `SHARD_TOTAL=16` (matrix) and optional `DATE_FOLDER` (defaults to today `YYYY-MM-DD`)
- Uses **manifest-first strategy**: single `list_repo_tree` call for the date folder → saves `manifest.json` → shard workers stream exclusively via CDN (`resolve/main/`) with **zero HF API calls** during data load
- Downloads via HF CDN with **no Authorization header** → completely bypasses `/api/` rate limits
- Projects heterogeneous schemas to `{prompt, response}` only at parse time (avoids pyarrow CastError); skips files that cannot be projected
- Dedups via central md5 store (`lib/dedup.py`) and writes deterministic shard output:
  ```
  batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl
  ```
- Retries with exponential backoff on 429 (wait 360s) and 5xx; skips already-processed files via dedup + manifest mtime check
- Logs structured JSON for GitHub Actions matrix visibility

def hf_get_tree(date_folder: str) -> List[Dict[str, Any]]:
    """Single API call: list top-level files in date folder (non-recursive)."""

            path=date_folder,
            repo_type="dataset",
            recursive=False,
        )

    except Exception as exc:
        log.error(json.dumps({"event": "tree_failed", "error": str(exc)}))
