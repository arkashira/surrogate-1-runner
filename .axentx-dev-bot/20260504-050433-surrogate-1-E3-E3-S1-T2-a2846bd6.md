```ts
export interface Alert {
  id: string;

  link: string;

export interface Approval {

  message: string;

> **Why string?**  
> JSON can’t serialise `Date` objects. Storing timestamps as ISO strings keeps the API simple and the UI can convert them to `Date` when needed.
