| **Configuration is externalised** | All knobs are read from environment variables (with sensible defaults) and optionally from a YAML file (`notification.yaml`). No hard‑coded secrets. |
| **Observability** | Structured JSON logs, a Prometheus‑compatible metric (`api_signature_changes_total`) and a small health‑check endpoint are included. |
