## Final Synthesis (Best Parts + Correctness + Actionability)

**Core diagnosis (merged, de-duplicated)**
- Backend ingestion still relies on authenticated HF API calls (`list_repo_tree`, `load_dataset`) that intermittently hit 429s and waste quota on re-enumeration.
- No persisted, deterministic file manifest; every run re-lists repos instead of using a cached list + CDN-only fetches.
- Mixed-schema parquet files are uploaded with extra columns (`source`, `ts`) instead of strict `{prompt, response}` projection, risking downstream schema errors.
- No retry/backoff for CDN fetches; transient CDN failures bubble up as hard crashes.
- Missing deterministic repo-to-sibling mapping for writes → cannot spread writes across sibling repos to bypass HF’s 128/hr commit cap.
- Training may silently die if the Lightning Studio stops between iterations (no idle-stop/heartbeat handling).

**Proposed change (merged, prioritized)**
- Add a backend manifest generator + CDN-only data loader with schema projection, retry/backoff, and local cache fallback.
- Add deterministic repo-to-sibling mapping and write-spreading helper to bypass commit-rate limits.
- Add small orchestration guard to survive Studio idle-stop (heartbeat + resume-from-last-checkpoint).
- Modify training entrypoint to use CDN loader + manifest cache and avoid authenticated API calls during training.

```python
# /opt/axentx/surrogate-1/backend/src/data/cdn_manifest.py
from __future__ import annotations

import json
import os
from datetime import datetime, timezone
from pathlib import Path
from typing import Dict, List, Optional

from huggingface_hub import HfApi

HF_REPO = os.getenv("HF_DATASET_REPO", "datasets/surrogate-1-mirror")
MANIFEST_TTL_SECONDS = int(os.getenv("MANIFEST_TTL_SECONDS", "86400"))  # 24h
SIBLINGS = json.loads(os.getenv("HF_SIBLINGS", "[]"))  # e.g. ["datasets/surrogate-1-mirror", ...]

api = HfApi()

def list_date_folder(date_folder: str, out_path: Path) -> List[str]:
    """
    Single API call to list files in date_folder (non-recursive).
    Emits JSON manifest to out_path.
    Returns list of relative file paths.
    """
    items = api.list_repo_tree(
        repo_id=HF_REPO,
        path=date_folder,
        repo_type="dataset",
        recursive=False,
    )
    files = sorted([it.rfilename for it in items if it.type == "file"])
    manifest = {
        "repo": HF_REPO,
        "date_folder": date_folder,
        "generated_utc": datetime.now(timezone.utc).isoformat(),
        "files": files,
    }
    out_path.parent.mkdir(parents=True, exist_ok=True)
    out_path.write_text(json.dumps(manifest, indent=2))
    return files

def load_manifest(manifest_path: Path) -> Dict:
    if not manifest_path.exists():
        raise FileNotFoundError(f"Manifest missing: {manifest_path}")
    data = json.loads(manifest_path.read_text())
    # Simple TTL check
    gen = datetime.fromisoformat(data["generated_utc"].replace("Z", "+00:00"))
    age = (datetime.now(timezone.utc) - gen).total_seconds()
    if age > MANIFEST_TTL_SECONDS:
        raise ValueError(f"Manifest expired ({int(age)}s > {MANIFEST_TTL_SECONDS}s)")
    return data

def pick_write_repo(file_key: str) -> str:
    """
    Deterministic repo-to-sibling mapping to spread writes and bypass HF commit cap.
    If no siblings configured, returns primary HF_REPO.
    """
    if not SIBLINGS:
        return HF_REPO
    idx = hash(file_key) % len(SIBLINGS)
    return SIBLINGS[idx]
```
