| **File‑system handling** | `os.makedirs(..., exist_ok=True)` | Switch to `pathlib.Path` – safer, more expressive, works on Windows & POSIX the same way. |
| **Logging** | `logging.basicConfig(level=logging.INFO)` in each file | Centralise logging configuration in the *application* entry‑point, not in a library. The modules now only obtain a logger via `logging.getLogger(__name__)`. |
