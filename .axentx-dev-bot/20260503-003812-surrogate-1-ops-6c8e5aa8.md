## Final Implementation Plan (≤2h)

**Highest-value change**: Add a Mac-side `tools/snapshot_manifest.py` that lists one date-partition via a single HF API call, emits `file_manifest.json` with CDN URLs + integrity metadata, and patch the Lightning training script to use a zero-API `IterableDataset` that streams parquet files directly from the CDN.

### Why this wins
- **Eliminates HF API rate-limits (429)** during training by replacing per-batch API calls with CDN downloads.
- **Single API call from Mac** (outside the training loop) fits safely within the 1000/5min quota.
- **CDN tier has much higher limits** and requires no auth for public datasets.
- **Reproducible runs** via pinned manifests and deterministic file ordering.
- **Minimal footprint**: no infra changes, no quota impact, and fits existing orchestration patterns.

#!/usr/bin/env python3
"""
Generate CDN-only manifest for a date partition in axentx/surrogate-1-training-pairs.

Usage:
    python tools/snapshot_manifest.py --repo axentx/surrogate-1-training-pairs \
        --date 2026-04-29 --out manifests/2026-04-29_manifest.json
"""

import argparse
import json
from pathlib import Path
from typing import Dict, List

from huggingface_hub import HfApi

HF_API = HfApi()

def build_manifest(repo: str, date_partition: str) -> List[Dict]:
    entries = HF_API.list_repo_tree(
        repo_id=repo,
        path=date_partition,
        recursive=True,
        repo_type="dataset",
    )

    manifest = []
    for entry in entries:
        if entry.type != "file":
            continue

        cdn_url = (
            f"https://huggingface.co/datasets/{repo}/resolve/main/{entry.path}"
        )
        item = {
            "repo": repo,
            "path": entry.path,
            "size": entry.size,
            "cdn_url": cdn_url,
        }

        # Include LFS metadata when available
        if getattr(entry, "lfs", None):
            item["lfs"] = {
                "oid": entry.lfs.get("oid"),
                "size": entry.lfs.get("size"),
            }
        manifest.append(item)

    # Deterministic ordering
    manifest.sort(key=lambda x: x["path"])
    return manifest
