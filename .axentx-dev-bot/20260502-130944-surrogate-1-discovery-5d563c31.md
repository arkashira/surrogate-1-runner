### Final Synthesis & Action Plan

**Core diagnosis (unified):**
- No persisted, date-scoped manifest → repeated HF repo enumeration → 429 rate limits and non-deterministic runs.
- Schema heterogeneity across repos → pyarrow `CastError` during training loads.
- HF API used instead of CDN → excess API calls and rate-limit risk.
- No projection to `{prompt, response}` before upload → mixed-schema files in enriched directory.
- No Lightning Studio idle-stop handling → lost work and quota waste.
- Missing systematic rate-limit/commit-cap handling.

**Chosen strategy (correctness + actionability):**
- Persist a date-scoped manifest once per date/repo, reuse it, and use CDN URLs for all downloads.
- Enforce `{prompt, response}` projection at ingestion time.
- Reuse running Studio instances; fall back to deterministic creation.
- Add lightweight retry/backoff for HF CDN calls and explicit schema enforcement.

```python
# hf_api_wrapper.py
import json
import os
import time
from datetime import datetime
from typing import List, Optional
import requests
from requests.adapters import HTTPAdapter, Retry

HF_CDN_BASE = "https://huggingface.co/datasets"

_SESSION: Optional[requests.Session] = None

def _get_session() -> requests.Session:
    global _SESSION
    if _SESSION is None:
        _SESSION = requests.Session()
        retries = Retry(
            total=5,
            backoff_factor=1,
            status_forcelist=[429, 500, 502, 503, 504],
            allowed_methods=["GET", "HEAD"],
        )
        _SESSION.mount("https://", HTTPAdapter(max_retries=retries))
    return _SESSION

def list_repo_files_via_cdn(repo: str, date_folder: str) -> List[str]:
    """
    List files under datasets/<repo>/resolve/main/<date_folder>/ using HF CDN.
    Returns filenames relative to date_folder.
    """
    session = _get_session()
    url = f"{HF_CDN_BASE}/{repo}/resolve/main/{date_folder}/"
    resp = session.get(url, timeout=30)
    resp.raise_for_status()

    # Naive HTML parsing for hrefs; robust for public HF CDN directory listing.
    # If repo uses LFS or custom layout, adjust accordingly.
    files = []
    for line in resp.text.splitlines():
        if 'href="' in line:
            href = line.split('href="', 1)[1].split('"', 1)[0].strip()
            # href is like "file.json" or "subdir/file.json"
            if href and not href.startswith("?") and not href.endswith("/"):
                files.append(href)
    return files

def build_and_cache_manifest(repo: str, date_folder: str, cache_dir: str = "manifests") -> List[str]:
    os.makedirs(cache_dir, exist_ok=True)
    cache_path = os.path.join(cache_dir, f"manifest_{repo.replace('/', '_')}_{date_folder}.json")

    if os.path.isfile(cache_path):
        with open(cache_path, "r") as f:
            return json.load(f)

    files = list_repo_files_via_cdn(repo, date_folder)
    with open(cache_path, "w") as f:
        json.dump(files, f, indent=2)
    return files

def cdn_download_url(repo: str, date_folder: str, filename: str) -> str:
    return f"{HF_CDN_BASE}/{repo}/resolve/main/{date_folder}/{filename}"
```
