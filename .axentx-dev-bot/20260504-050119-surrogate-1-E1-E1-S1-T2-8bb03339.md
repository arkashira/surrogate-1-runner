## Unified, production‑ready LLM provider layer

Below is a single, self‑contained Java module that

* **Supports all three major APIs** (OpenAI, Anthropic Claude, Google Gemini) through a single `LLMProvider` interface.
* Uses **Java 11+ HttpClient** (no external HTTP client) and **Jackson** for JSON.
* Handles **environment‑based configuration** with sensible defaults.
* Provides **robust error handling** (status‑code checks, JSON‑structure validation, time‑outs).
* Is **easy to extend** – just add a new class that implements `LLMProvider`.

```java
// ────────────────────────────────────────────────────────────────────────────────
// src/main/java/com/axentx/llm/LLMProvider.java
// ────────────────────────────────────────────────────────────────────────────────
package com.axentx.llm;

import java.io.IOException;

/**
 * Unified contract for all LLM providers.
 */
public interface LLMProvider {
    /**
     * Generates text for the given prompt.
     *
     * @param prompt the input prompt
     * @return the generated text
     * @throws IOException          if a network or parsing error occurs
     * @throws InterruptedException if the request is interrupted
     */
    String generateText(String prompt) throws IOException, InterruptedException;
}
```

package com.axentx.llm;

import java.net.http.HttpClient;
import java.time.Duration;
import com.fasterxml.jackson.databind.ObjectMapper;

/**
 * Base class that gives every provider a shared HttpClient and ObjectMapper.
 * Also centralises common configuration logic.
 */
public abstract class AbstractProvider implements LLMProvider {

    protected final HttpClient httpClient;
    protected final ObjectMapper objectMapper;
    protected final String apiKey;
    protected final String model;
    protected final Duration timeout = Duration.ofSeconds(30);

    protected AbstractProvider(String keyEnv, String modelEnv, String defaultModel) {
        this.httpClient = HttpClient.newBuilder()
                .connectTimeout(Duration.ofSeconds(10))
                .build();
        this.objectMapper = new ObjectMapper();

        this.apiKey = System.getenv(keyEnv);
        if (apiKey == null || apiKey.isBlank()) {
            throw new IllegalStateException(keyEnv + " environment variable is required");
        }

        this.model = Optional.ofNullable(System.getenv(modelEnv)).orElse(defaultModel);
    }
}
