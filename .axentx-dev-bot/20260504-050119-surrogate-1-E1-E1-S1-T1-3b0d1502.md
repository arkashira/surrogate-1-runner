## Unified, Provider‑Agnostic LLM Layer  
*(A single, production‑ready Java API that works with OpenAI, Anthropic, Google, and any future provider.)*

---

### 1. Core Design Principles

| Principle | Why it matters | How we implement it |
|-----------|----------------|---------------------|
| **Single responsibility** | Keeps the gateway thin and testable. | `LLMProvider` only knows how to *generate* a string. |
| **Provider‑agnostic** | Switching providers is a one‑liner. | `LlamaGateway` delegates to a registered `LLMProvider`. |
| **Fail‑fast configuration** | Avoids silent failures at runtime. | Constructors validate required env‑vars and throw `LLMException`. |
| **Graceful error handling** | Consumers get a consistent exception type. | All provider code throws `LLMException`. |
| **Extensibility** | New providers need only implement the interface. | `LLMProvider` is an interface; no changes to the gateway. |
| **Thread‑safety** | Gateway is used in concurrent services. | `ConcurrentHashMap` for providers; stateless provider implementations. |
| **Minimal external dependencies** | Keeps the repo lightweight. | Uses only JDK 17 `java.net.http`. |

---

### 2. Package Layout

```
src/main/java/com/axentx/llm/
 ├─ LlamaGateway.java
 ├─ LLMProvider.java
 ├─ LLMException.java
 ├─ provider/
 │   ├─ OpenAIProvider.java
 │   ├─ AnthropicProvider.java
 │   └─ GoogleProvider.java   (optional)
 └─ model/
     ├─ ChatRequest.java
     ├─ ChatResponse.java
     ├─ CompletionRequest.java
     └─ CompletionResponse.java
```

*`model` classes are simple POJOs that mirror the JSON payloads of the providers.*

---

### 3. Core Interface

```java
package com.axentx.llm;

/**
 * Minimal abstraction for a Large Language Model provider.
 * Implementations hide provider‑specific request/response handling.
 */
public interface LLMProvider {
    /**
     * Generate a completion for the given prompt.
     *
     * @param prompt the user prompt
     * @return the generated text
     * @throws LLMException if the provider call fails
     */
    String generate(String prompt) throws LLMException;
}
```

---

### 4. Unified Exception

```java
package com.axentx.llm;

public class LLMException extends RuntimeException {
    public LLMException(String message, Throwable cause) { super(message, cause); }
    public LLMException(String message) { super(message); }
}
```
