| **Testing & CI** | None | Must be testable | Provide **unit tests** for handler logic, **integration test** using `httptest`, and a **GitHub Actions** workflow that builds, lints, and runs tests on every push. |
| **Deployment** | Binary run on port 8080 | Works locally | Add a **Dockerfile** (multi‑stage build) and a **Kubernetes manifest** (Deployment + Service + ConfigMap for limits). |
| **Configuration** | Hard‑coded values | Inflexible | Use **Viper** to read env vars / config file for port, max upload size, processing timeout, and log level. |
