## Unified design – “surrogate‑1” drift‑event persistence layer  

Below is a single, coherent implementation that merges the strongest ideas from the two proposals:

* **A single, thread‑safe PostgreSQL connection pool** (`db.Init`) that reads its configuration from environment variables (easy for CI/CD) **and** also lets the caller supply an explicit DSN when they need more control.  
* **A clean, reusable `model.DriftEvent` struct** that:
  * uses clear field names (`ID`, `DetectedAt`, `Service`, `DiffDetails`);
  * stores the JSON diff as `json.RawMessage` (so callers can pass either a pre‑marshalled `[]byte` or any Go value);
  * carries struct tags for both JSON marshaling and the `sqlx`/`database/sql` mapper (`db:"…"`);
* **Two‑layer data‑access API**:
  * a low‑level `*sql.DB` accessor (`db.Get`) for code that already works with `database/sql`;
  * a higher‑level wrapper type `db.PG` that implements the concrete operation `InsertDriftEvent(ctx, *model.DriftEvent) (int64, error)`.  
  This wrapper adds context‑support, automatic timestamp filling, and flexible JSON handling (the “switch‑on‑type” logic from candidate 2).  
* **Convenient constructor** `model.NewDriftEvent(service, diffObj)` that marshals any Go value into JSON and sets `DetectedAt` to `time.Now().UTC()`.  

The result is a **single, import‑friendly package layout** that can be used both in simple scripts (`db.Init` + `model.DriftEvent.Insert`) **and** in larger services that prefer a richer repository object (`db.New(...).InsertDriftEvent`).

import (
	"context"
	"database/sql"
	"encoding/json"
	"fmt"
	"os"
	"sync"
	"time"

	_ "github.com/lib/pq" // PostgreSQL driver

	once sync.Once
	pool *sql.DB
)

// Init creates (once) a *sql.DB pool using environment variables.
// It is safe for concurrent callers and returns the same instance every time.
func Init() (*sql.DB, error) {
	var err error
	once.Do(func() {
		// Environment‑based configuration – works out‑of‑the‑box in most containers.
		host := getenv("PGHOST", "localhost")
		port := getenv("PGPORT", "5432")
		user := getenv("PGUSER", "postgres")
		password := getenv("PGPASSWORD", "")
		dbname := getenv("PGDATABASE", "surrogate")
		sslmode := getenv("PGSSLMODE", "disable") // allow override

		dsn := fmt.Sprintf(
			"host=%s port=%s user=%s password=%s dbname=%s sslmode=%s",
			host, port, user, password, dbname, sslmode,
		)

		pool, err = sql.Open("postgres", dsn)
		if err != nil {
			return
		}
		// Reasonable defaults – adjust as needed.
		pool.SetMaxOpenConns(25)
		pool.SetMaxIdleConns(5)
		pool.SetConnMaxLifetime(5 * time.Minute)

		// Verify the connection early.
		if pingErr := pool.Ping(); pingErr != nil {
			err = fmt.Errorf("postgres ping failed: %w", pingErr)
			return
		}
	})
	return pool, err
}

// Get returns the already‑initialized *sql.DB.
// Panics if Init has not been called successfully – this mirrors the behaviour
// of many popular singleton helpers and surfaces programmer error early.
func Get() *sql.DB {
	if pool == nil {
		panic("database not initialized; call db.Init() first")
	}
	return pool
}

// caller wants to manage multiple connections or avoid the env‑var defaults.

	db, err := sql.Open("postgres", dsn)
	if err != nil {
		return nil, fmt.Errorf("open postgres: %w", err)
	}
	if err = db.Ping(); err != nil {
		return nil, fmt.Errorf("ping postgres: %w", err)
	}
	return &PG{db: db}, nil
}

type PG struct {
