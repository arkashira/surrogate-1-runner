## 1. Acceptance Criteria
- **AC1** – The parser must locate every pair of markers `<!--DOC-START:<id>-->` and `<!--DOC-END-->` in a markdown file and return their start line, end line, and `id`.
- **AC2** – When a function annotated with `@doc(id="<id>")` is detected as changed, the parser’s `replace_section(id, snippet)` method must replace the entire block (including markers) with the new snippet wrapped in the same start/end markers.
- **AC3** – If `replace_section` is called with an `id` that does not exist in the file, the parser must append a new block at the end of the file in the form:  
  `<!--DOC-START:<id>-->\n<snippet>\n<!--DOC-END-->\n`
- **AC4** – After any modification, the file must end with exactly one newline character.
- **AC5** – The `DocSync` command must create a new git branch named `doc-sync/<commit‑sha>` (where `<commit‑sha>` is the SHA of the current HEAD) and push a PR targeting the original source branch.
- **AC6** – All operations must be idempotent: running the sync twice without further code changes must result in no new commits or PRs.
- **AC7** – Errors (malformed markers, missing end marker, duplicate IDs) must raise a `DocSyncError` with a clear message and must not create a branch or PR.

import pytest
from surrogate_1.doc_sync.parser import MarkdownParser, DocSyncError

SAMPLE_MD = """# Title

<!--DOC-START:foo-->
Old content for foo
<!--DOC-END-->

Some other text

<!--DOC-START:bar-->
Old content for bar
<!--DOC-END-->
"""

def test_detect_all_markers():
    parser = MarkdownParser(content=SAMPLE_MD)
    blocks = parser.find_blocks()
    assert len(blocks) == 2
    assert blocks[0].id == "foo"
    assert blocks[1].id == "bar"
    # verify line numbers
    assert blocks[0].start_line == 3
    assert blocks[0].end_line == 5

def test_replace_existing_section():
    parser = MarkdownParser(content=SAMPLE_MD)
    new_snippet = "New **foo** docs"
    updated = parser.replace_section("foo", new_snippet)
    expected = """# Title

<!--DOC-START:foo-->
New **foo** docs
<!--DOC-END-->

Some other text

<!--DOC-START:bar-->
Old content for bar
<!--DOC-END-->
"""
    assert updated == expected
    # ensure file ends with single newline
    assert updated.endswith("\n") and not updated.endswith("\n\n")

def test_append_when_id_missing():
    parser = MarkdownParser(content=SAMPLE_MD)
    new_snippet = "Docs for baz"
    updated = parser.replace_section("baz", new_snippet)  # should append
    assert "<!--DOC-START:baz-->" in updated
    assert "Docs for baz" in updated
    # appended block must be last before final newline
    assert updated.rstrip().endswith("<!--DOC-END-->")
    assert updated.endswith("\n")

<!--DOC-END-->
