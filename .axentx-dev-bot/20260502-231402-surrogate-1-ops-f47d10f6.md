### 1) Snapshot repo tree → JSON (Mac orchestrator) — 20–30 min
- Script: `bin/list-snapshot.sh` (preferred) or `bin/list-snapshot.py`  
- Uses `huggingface_hub.list_repo_tree(repo_id, path="public-merged/YYYY-MM-DD", recursive=False)`  
- Saves `snapshot-{date}.json` with minimal, CDN-ready entries:
  ```json
  {
    "repo": "datasets/axentx/surrogate-1-training-pairs",
    "date": "2026-05-02",
    "files": [
      "public-merged/2026-05-02/file1.parquet",
      "public-merged/2026-05-02/file2.parquet"
    ]
  }
  ```
- Rate-limit safe: one tree call per folder; non-recursive to avoid pagination.  
- Retry on 429 with exponential backoff (cap ~360 s) and exit non-zero on persistent failure.

**Concrete snippet (bash):**  
```bash
#!/usr/bin/env bash
set -euo pipefail
REPO="axentx/surrogate-1-training-pairs"
DATE="${1:-$(date +%Y-%m-%d)}"
OUT="snapshot-${DATE}.json"

python3 - "$REPO" "$DATE" "$OUT" <<'PY'
import json, sys
from huggingface_hub import HfApi

repo_id, date_folder, out_path = sys.argv[1], sys.argv[2], sys.argv[3]
api = HfApi()
tree = api.list_repo_tree(repo_id, path=f"public-merged/{date_folder}", recursive=False)
files = sorted(f.rfilename for f in tree if f.rfilename.endswith(".parquet"))
with open(out_path, "w") as f:
    json.dump({"repo": repo_id, "date": date_folder, "files": files}, f, indent=2)
print(f"Snapshot saved: {out_path} ({len(files)} files)")
PY
```

### 2) Training script: CDN-only ingestion — 40–60 min
- New script or flag: `train-cdn.py` / `--file-list snapshot-YYYY-MM-DD.json`  
- Data loader reads local JSON and streams via raw CDN URLs:  
  `https://huggingface.co/datasets/axentx/surrogate-1-training-pairs/resolve/main/{path}`  
- No `load_dataset`, no `list_repo_files`, no `hf_hub_download` during training → zero API calls.  
- Stream to temp file to avoid memory blow-up; use `pyarrow.parquet` or line-iterate JSONL as needed.  
- Project schema to `{prompt, response}` at parse time; drop malformed rows.

**Concrete snippet (Python):**
```python
import json, argparse, tempfile
from pathlib import Path
import pyarrow.parquet as pq
import requests

HF_REPO = "axentx/surrogate-1-training-pairs"

def cdn_url(path: str) -> str:
    return f"https://huggingface.co/datasets/{HF_REPO}/resolve/main/{path}"

def stream_parquet(path: str):
    url = cdn_url(path)
    with requests.get(url, stream=True, timeout=30) as r:
        r.raise_for_status()
        tmp = Path(tempfile.mktemp(suffix=".parquet"))
        with open(tmp, "wb") as f:
            for chunk in r.iter_content(chunk_size=8192):
                f.write(chunk)
        return pq.read_table(tmp).to_pylist()

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--file-list", required=True, help="snapshot-YYYY-MM-DD.json")
    args = parser.parse_args()

    with open(args.file_list) as f:
        files = json.load(f)["files"]

    pairs = []
    for fpath in files:
        print(f"CDN fetch: {fpath}")
        rows = stream_parquet(fpath)
        for r in rows:
            if r.get("prompt") and r.get("response"):
                pairs.append({"prompt": r["prompt"], "response": r["response"]})
    print(f"Total pairs: {len(pairs)}")
    # continue training...

if __name__ == "__main__":
    main()
```
