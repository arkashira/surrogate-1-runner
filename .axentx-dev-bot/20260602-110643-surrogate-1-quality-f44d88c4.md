### **Integration tests**
#### Happy path
1. `GET /health` → `200 OK` with `{"status":"ok"}`
2. `GET /health/` (trailing slash) → `200 OK` with `{"status":"ok"}`
3. `GET /health` with query params (e.g., `?foo=bar`) → `200 OK` (ignores params)
4. `GET /health` from multiple clients concurrently → all return `200 OK`
5. `GET /health` after server restart → still returns `200 OK`

#### Edge cases
1. `GET /Health` (wrong case) → `404 Not Found`
2. `POST /health` → `405 Method Not Allowed`
