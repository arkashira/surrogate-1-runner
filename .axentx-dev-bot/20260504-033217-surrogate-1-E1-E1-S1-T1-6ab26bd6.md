## 1. Acceptance Criteria  

- **Capacity**  
  - `BatteryReader.read_capacity()` returns an integer between 0 % and 100 % inclusive.  
  - The value matches the content of `/sys/class/power_supply/BAT*/capacity` (rounded to nearest integer).  

- **Wear**  
  - `BatteryReader.read_wear()` returns a float between 0 % and 100 % inclusive.  
  - The value is calculated as `1 - (cycle_count / (charge_full_design / 100))`.  
  - When `cycle_count` or `charge_full_design` are missing or zero, the method raises `BatteryReaderError`.  

- **Temperature**  
  - `BatteryReader.read_temperature()` returns a float in degrees Celsius.  
  - The value matches the content of `/sys/class/power_supply/BAT*/temp` divided by 1000 (sysfs reports in millidegrees).  

- **Polling Interval**  
  - `BatteryReader.start_polling(callback)` invokes `callback(metrics)` every 5 seconds.  
  - The callback receives a `BatteryMetrics` instance with updated values.  

- **Error Handling**  
  - Missing sysfs files or unreadable content cause a logged warning and the metric is set to `None`.  
  - All exceptions are caught; the reader continues polling.
