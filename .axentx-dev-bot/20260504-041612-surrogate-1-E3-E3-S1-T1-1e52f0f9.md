└───────────────────────┘   └───────────────────────┘

* **InstanceDatabase** – a JSON/YAML file that contains the full spec of every instance type (vCPU, memory, cost, storage, network).  
* **RecommendationEngine** – applies thresholds, cost‑per‑vCPU, cost‑per‑GB‑memory, and a simple “confidence” model.  
* **CostCalculator** – can be extended to use RI/Spot pricing; for now it uses on‑demand.  
* **RecommendationWriter** – serialises the recommendation to JSON, writes to S3, or posts to SNS.
