## 1. Acceptance Criteria
- **AC1 – Dashboard Availability**: The analytics dashboard is reachable at `/dashboard/analytics` and returns HTTP 200 within 200 ms for an authenticated freelancer user.
- **AC2 – Completion Rate Metric**: The dashboard displays a numeric “Task Completion Rate” calculated as `completed_tasks / total_assigned_tasks * 100` with precision to two decimal places.
- **AC3 – Time‑Spent Metric**: The dashboard shows “Average Time Spent per Task” in minutes, derived from the sum of `time_spent` on completed tasks divided by the number of completed tasks.
- **AC4 – Report Generation**: A “Generate Report” button produces a PDF/CSV file containing the displayed metrics and a list of tasks with status, timestamps, and time spent.
- **AC5 – Export Functionality**: The generated report can be downloaded via a `GET /reports/{report_id}/download` endpoint, returning the correct `Content‑Type` and file size > 0 KB.
- **AC6 – Role‑Based Access**: Only users with the `freelancer` role can access the dashboard; other roles receive HTTP 403.
- **AC7 – Data Refresh**: Metrics on the dashboard automatically refresh every 5 minutes without a full page reload (via WebSocket or polling), and the refreshed values differ from the previous snapshot when underlying data changes.
