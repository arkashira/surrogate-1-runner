### 1. Acceptance Criteria

- **[AC-1]** A 'Launch Terminal' button is rendered on the node detail page for all nodes where `node.os === 'linux'`
- **[AC-2]** Clicking the button opens a full-screen xterm.js terminal overlay within 500ms of click
- **[AC-3]** WebSocket connection to `/api/v1/nodes/{nodeId}/terminal` is established with TLS within 2000ms
- **[AC-4]** On connection failure, an error message "Unable to connect to terminal. [Retry]" is displayed within 1000ms
- **[AC-5]** The terminal accepts keyboard input and displays output from the backend
- **[AC-6]** The terminal can be closed via ESC key or a close button, returning focus to the node detail page
