## 1. Acceptance Criteria
- **AC1 – Randomized User‑Agent**: Every outbound HTTP request must include a `User-Agent` header selected at random from a configurable list of ≥10 strings; the same request must never reuse the exact same UA within a 5‑minute window.
- **AC2 – Proxy IP Rotation**: The gateway must route each request through a proxy selected from a pool of ≥5 IPs, rotating on a per‑request basis; no two consecutive requests may use the same proxy IP.
- **AC3 – TLS 1.3 Enforcement**: All outbound connections must negotiate TLS 1.3; any fallback to TLS 1.2 or lower must cause the request to be aborted and logged as a failure.
- **AC4 – Custom Stealth Header**: Every request must contain the header `X‑Stealth‑Token: <sha256‑of‑payload>` (or a static placeholder in tests); missing or malformed header results in request rejection.
- **AC5 – HTTPS Only**: The gateway must reject any attempt to send traffic over plain HTTP; such attempts must be logged with a severity of `WARN`.
- **AC6 – Health‑Check Endpoint**: `GET /health/stealth` must return HTTP 200 with JSON `{ "status":"ok", "proxy":"<ip>", "ua":"<string>" }` within 200 ms, confirming that a test request passed all stealth checks.
- **AC7 – Observability**: Each request must emit a structured log entry containing `request_id`, `proxy_ip`, `user_agent`, `tls_version`, and `stealth_header_status` (valid/invalid).

func TestSelectProxy_RotatesIP(t *testing.T) {
    pool := []string{"10.0.0.1:8080", "10.0.0.2:8080", "10.0.0.3:8080"}
    stealth.SetProxyPool(pool)

    var last string
    for i := 0; i < 10; i++ {
        ip := stealth.SelectProxy()
        assert.Contains(t, pool, ip, "Selected proxy must belong to the pool")
        if i > 0 {
            assert.NotEqual(t, ip, last, "Proxy IP must rotate per request")
        }
        last = ip
    }
}
