- **Memory Footprint** – The parser must never allocate more than **500 MB** of RAM during a full parse of any input, regardless of file size.

- **Default Streaming** – When the input size is > 10 MB, the parser automatically switches to streaming mode.  
- **Chunk Size Validation** – `chunk_size` must be a positive integer; otherwise a `ValueError` is raised.  
- **Buffer Size Validation** – `max_buffer_size` must be a positive integer; otherwise a `ValueError` is raised.  
- **Source Validation** – `source` must be either a readable file path or a file‑like object; otherwise a `TypeError` is raised.  
- **File Opening Errors** – If a file path cannot be opened, an `IOError` (or `OSError`) is raised.
