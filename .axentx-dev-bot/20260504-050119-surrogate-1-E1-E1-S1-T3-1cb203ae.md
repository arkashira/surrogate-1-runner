## dev — claude/llm-fallback-chain @ 2026-05-16T04:28:13.517070Z

Below is a **single, production‑ready implementation** that pulls together the strongest ideas from the two candidates:

* **Unified interface** – `LLMProvider` is the contract that every provider implements.  
* **Configuration** – The provider reads its endpoint, API key and model from a central `LLMConfig` object (so the rest of the codebase can stay agnostic).  
* **Modern HTTP client** – Java 17’s `java.net.http.HttpClient` is used instead of the older Apache client.  
* **Robust JSON handling** – Jackson’s `JsonNode` gives us a safe way to navigate the response tree.  
* **Clear error handling** – All failures surface as a custom `LLMException` that carries the HTTP status, body and a friendly message.  
* **Testability** – The HTTP client is injected, so unit tests can provide a mock client that returns a canned response.  
* **Logging** – SLF4J is used for trace‑level debugging and error logging.

## 1.  The `LLMProvider` interface

```java
package com.axentx.llm;

/**
 * Every LLM provider must implement this interface.
 * The contract is intentionally minimal – it only needs a single
 * method that turns a request into a response.
 */
public interface LLMProvider {

    /**
     * Generate a completion for the given request.
     *
     * @param request the request payload
     * @return the provider‑specific response
     * @throws LLMException if the provider fails
     */
    LLMResponse generateResponse(LLMRequest request) throws LLMException;
}
```

import lombok.Builder;
import lombok.Value;

import java.util.Map;

@Value
@Builder
public class LLMResponse {
    String content;
    String provider;          // e.g. "minimax"
    Map<String, Object> metadata;   // raw response, timestamps, etc.
}

// LLMException – unchecked wrapper for provider errors
package com.axentx.llm;

public class LLMException extends RuntimeException {
    private final int statusCode;
    private final String responseBody;

    public LLMException(String message, int statusCode, String responseBody) {
        super(message);
        this.statusCode = statusCode;
        this.responseBody = responseBody;
    }

    public int getStatusCode() { return statusCode; }
    public String getResponseBody() { return responseBody; }
}
