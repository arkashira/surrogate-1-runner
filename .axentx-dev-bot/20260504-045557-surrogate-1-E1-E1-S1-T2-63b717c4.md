    direction: str

    Returns
    -------
    Draft7Validator
        A compiled validator ready for ``iter_errors``.

    Raises
    ------
    FileNotFoundError
        If the expected ``<job>_<direction>.json`` file does not exist.
    json.JSONDecodeError
