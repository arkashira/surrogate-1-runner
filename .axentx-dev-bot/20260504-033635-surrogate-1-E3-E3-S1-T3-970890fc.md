## 1. Acceptance Criteria
- **Hierarchical Support**: `WorkflowManagement` can create, store, and retrieve agents organized in a parent‑child tree up to any depth.
- **Pause/Resume**: Each agent (and its subtree) can be paused and later resumed, preserving execution state.
- **State Consistency**: Pausing an agent automatically pauses all descendant agents; resuming restores all to the correct state.
- **Query API**: `getAgentState(agentId)` returns `ACTIVE`, `PAUSED`, or `COMPLETED`.
- **Concurrency**: Concurrent pause/resume operations on different branches do not corrupt the hierarchy or state.
- **Performance**: Pausing or resuming a subtree of 1,000 agents completes in < 200 ms on a standard test machine.
- **Error Handling**: Pausing an already paused agent or resuming an active agent returns a clear error without altering state.

## 2. Unit Tests (pseudo‑code, Jest style)

```javascript
describe('WorkflowManagement', () => {
  let wf;

  beforeEach(() => {
    wf = new WorkflowManagement();
  });

  test('creates a root agent', () => {
    const root = wf.createAgent('root');
    expect(root.id).toBeDefined();
    expect(root.parent).toBeNull();
  });

  test('adds child agents to a parent', () => {
    const parent = wf.createAgent('parent');
    const child = wf.createAgent('child', parent.id);
    expect(child.parent.id).toBe(parent.id);
    expect(parent.children).toContain(child);
  });

  test('pauses an agent and its descendants', () => {
    const a = wf.createAgent('a');
    const b = wf.createAgent('b', a.id);
    const c = wf.createAgent('c', b.id);
    wf.pauseAgent(a.id);
    expect(wf.getAgentState(a.id)).toBe('PAUSED');
    expect(wf.getAgentState(b.id)).toBe('PAUSED');
    expect(wf.getAgentState(c.id)).toBe('PAUSED');
  });

  test('resumes an agent and its descendants', () => {
    const a = wf.createAgent('a');
    wf.pauseAgent(a.id);
    wf.resumeAgent(a.id);
    expect(wf.getAgentState(a.id)).toBe('ACTIVE');
  });

  test('pausing an already paused agent throws error', () => {
    const a = wf.createAgent('a');
    wf.pauseAgent(a.id);
    expect(() => wf.pauseAgent(a.id))
      .toThrow('Agent already paused');
  });

  test('resuming an active agent throws error', () => {
    const a = wf.createAgent('a');
    expect(() => wf.resumeAgent(a.id))
      .toThrow('Agent not paused');
  });

  test('concurrent pause/resume on different branches is safe', async () => {
    const a = wf.createAgent('a');
    const b = wf.createAgent('b');
    const c = wf.createAgent('c', a.id);
    const d = wf.createAgent('d', b.id);
    await Promise.all([
      wf.pauseAgent(a.id),
      wf.resumeAgent(b.id).catch(() => {}) // b is active, should error
    ]);
    expect(wf.getAgentState(a.id)).toBe('PAUSED');
    expect(wf.getAgentState(b.id)).toBe('ACTIVE');
  });
});
```
