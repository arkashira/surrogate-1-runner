# Java 8 Default Method Overrides for Synchronized/Concurrent Collections

## Default Method Overrides Examples

### 1. Overriding `forEach` in `Collections.synchronizedList`

```java
import java.util.Collections;
import java.util.List;
import java.util.ArrayList;

public class SynchronizedListExample {
    public static void main(String[] args) {
        List<String> synchronizedList = Collections.synchronizedList(new ArrayList<>());

        // Adding elements
        synchronizedList.add("Element 1");
        synchronizedList.add("Element 2");

        // Overriding forEach to ensure thread safety
        synchronizedList.forEach(element -> {
            System.out.println(element);
        });
    }
}

```java
